{[%hh_hdr_ru:Окно свойств изображения :: Данные]} {[%hh_h2:Данные изображений]}

Эта страница диалога свойств позволяет поменять основную часть изменяемых данных изображений:

  • Дату — дата изображения. Формат даты определяется региональными настройками Windows. Если данных о дате изображения нет, это поле не заполнено.
  • Время — время изображения. Если данных о времени нет, оставьте поле пустым.
  • Место — место изображённых событий. Выпадающий список отображает все имеющиеся в фотоальбоме варианты.
  • Номер/название плёнки — номер, присвоенный плёнке при проявке, или данное ей Вами название. Выпадающий список отображает все имеющиеся в фотоальбоме варианты.
  • Номер кадра — без комментариев.
  • Автор — имя автора или фотографа. Выпадающий список отображает все имеющиеся в фотоальбоме варианты.
  • Описание — краткое описание изображения, используемое для отображения при просмотре.
  • Примечание — все данные, не попавшие в остальные поля.
  • Наименование/код носителя — название носителя, на котором расположен файл изображения (например, номер компакт-диска). Выпадающий список отображает все имеющиеся в фотоальбоме варианты.

Более подробная информация о данных изображения находится здесь. {[%hh_h2:Несколько изображений]}

PhoA поддерживает редактирование данных для любого числа изображений одновременно. Если говорить о текущей странице, то здесь используется незамысловатая схема обозначения свойств:

  • Если какое-то свойство совпадает у всех выделенных изображений, соответствующее поле ввода будет белого цвета (точнее, цвета окна в Windows).
  • Если значения определённого свойства различаются у разных изображений, соответствующее поле закрашивается в серый цвет (а точнее, в цвет рельефных объектов Windows). Если Вы начнёте изменять данные в этом поле, то у всех выделенных изображений соответствующее свойство примет введённое Вами значение, и поле ввода перекрасится в белый цвет. В противном случае эти данные останутся неизменными (и, соответственно, различными).

Впоследствии по введённым Вами данным можно осуществлять поиск и выборку изображений, поэтому стоит ответственно отнестись к тому, чего вы там вводите. {[%hh_undosup_ru]} {[%hh_seealso_ru]} Данные изображений
