Мастер операций с файлами изображений :: Операция перемещения файлов изображений

Эта операция в целом идентична операции копирования файлов, за исключением двух моментов:

  • Ссылки на файлы в изображениях фотоальбома в процессе перемещения исправляются на новые.
  • По выбору пользователя эта операция позволяет удалить исходные файлы: совсем или в Корзину.

Операция перемещения полезна, если Вы хотите реорганизовать файлы в Вашем фотоальбоме, например, перенести их в другую папку или на другой диск.

Чтобы выполнить эту операцию, необходимо запустить Мастер операций с файлами изображений и на странице выбора задачи выбрать Переместить файлы изображений в папку.

Далее последовательность страниц такова:

  • Выбор изображений для операции
  • Выбор папки назначения
  • Опции копирования/перемещения
  • Опции создания CD/DVD
  • Процесс выполнения операции
  • Протокол операции
