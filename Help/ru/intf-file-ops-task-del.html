{[%hh_hdr_ru:Мастер операций с файлами изображений :: Операция удаления файлов изображений]}

Эта операция позволяет удалить ненужные файлы изображений (естественно, вместе с соответствующими изображениями, так как иначе эти изображения ссылались бы на несуществующие файлы). Мастер позволяет удалить файлы изображений совсем или переместить их в Корзину.

Чтобы выполнить эту операцию, необходимо запустить Мастер операций с файлами изображений и на странице выбора задачи выбрать Удалить изображения и файлы.

Далее последовательность страниц такова:

  • Выбор изображений для операции
  • Опции удаления
  • Процесс выполнения операции
  • Протокол операции
