{[%hh_hdr_ru:Режим обзора :: Меню "Вид"]}

Это меню контролирует вид окна программы:

  • Показ панели инструментов задаётся пунктом Вид | Панель инструментов.
  • Показ строки состояния определяется пунктом Вид | Строка состояния.
  • {[%hh_ii:RemoveSearchResults]} Пункт Вид | Убрать результаты поиска скрывает папку результатов поиска.
  • Далее следуют пункты для работы с представлениями фотоальбома:
    • {[%hh_ii:Folder]} Пункт Группы изображений служит для перехода к основному режиму просмотра дерева папок, когда в левом окне программы отображаются группы фотоальбома (режим по умолчанию).
    • {[%hh_ii:NewView]} Пункт Новое представление... открывает окно свойств для создания нового представления.
    • {[%hh_ii:Delete]} Пункт Удалить представление удаляет текущее представление (если в данный момент активно именно представление).
    • {[%hh_ii:Edit]} Пункт Изменить представление... открывает окно свойств для редактирования текущего представления (если в данный момент активно именно представление).
    • {[%hh_ii:GroupFromView]} Пункт Поместить в группу... вызывает диалоговое окно выбора группы фотоальбома, позволяющее создать копию папок представления в группах фотоальбома.
