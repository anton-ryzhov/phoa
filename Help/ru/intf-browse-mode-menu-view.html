{[%hh_hdr_ru:Режим обзора :: Меню "Вид"]}

Это меню контролирует вид окна программы.
Команда Описание
Панель инструментов Управляет отображением основной панели инструментов.
Строка состояния Управляет отображением строки состояния.
Рекурсивный просмотр Когда включен режим рекурсивного просмотра, в окне просмотра эскизов отображаются эскизы не только из группы, выбранной в дереве, а также из всех её подгрупп. Переупорядочивание эскизов мышью в этом режиме допускается только если текущая группа не содержит подгрупп.
{[%hh_ii:RemoveSearchResults]} Убрать результаты поиска Скрывает папку результатов поиска.
Далее следуют пункты для работы с представлениями фотоальбома
{[%hh_ii:Folder]} Группы изображений Служит для перехода к основному режиму просмотра дерева папок, когда в окне групп программы отображаются группы фотоальбома (режим по умолчанию).
Далее следуют пункты представлений фотоальбома. При выборе пункта активизируется соответствующее представление, и в дереве групп отображается автоматически сгенерированная программой иерархия его групп.
{[%hh_ii:NewView]} Новое представление... Открывает окно свойств представления для создания нового представления.
{[%hh_ii:Delete]} Удалить представление Удаляет текущее представление (если в данный момент активно именно представление).
{[%hh_ii:Edit]} Изменить представление... Открывает окно свойств для редактирования текущего представления (если в данный момент активно именно представление).
{[%hh_ii:GroupFromView]} Поместить в группу... Вызывает диалоговое окно выбора группы фотоальбома, позволяющее создать копию папок представления в группах фотоальбома.
