{[%hh_hdr_ru:Инструменты]}

Инструмент в PhoA — это определяемая (и настраиваемая) пользователем команда. Инструменты обладают следующими характеристиками:

  • Они бывают разного вида.
  • Они применимы к файлам, подходящим под заданные маски.
  • Они отображаются в меню; в каких именно — определяется установленными флагами использования инструмента.
  • Они доступны, только если есть выделенные изображения.
  • Они имеют наименование и подсказку (кроме пункта-разделителя). Наименование отображается в меню, а подсказка — в строке состояния при выборе пункта меню.
{[%hh_h2:Виды инструментов]}

Текущая версия PhoA поддерживает следующие виды инструментов:

  • Пункт-разделитель
  • Действие по умолчанию
  • Стандартное действие "Открыть"
  • Стандартное действие "Изменить"
  • Стандартное действие "Печать"
  • Заданное действие

Пункт-разделитель

Представляет собой статический разделитель в меню.

Действие по умолчанию

Выполняет действие, которое по умолчанию установлено для типа файла, к которому инструмент применяется.

Стандартное действие "Открыть"

Выполняет действие Открыть (open), которое установлено для типа файла, к которому инструмент применяется.

Стандартное действие "Изменить"

Выполняет действие Изменить (edit), которое установлено для типа файла, к которому инструмент применяется.

Стандартное действие "Печать"

Выполняет действие Печать (print), которое установлено для типа файла, к которому инструмент применяется.

NB: Действие по умолчанию и стандартные действия задаются с помощью диалога Свойства папки меню Сервис Проводника Windows (закладка Типы файлов).

Заданное действие

Вы сами задаёте приложение и параметры его вызова в Диалоге свойств инструмента. {[%hh_h2:Маски файлов]}

Инструмент становится доступным (или отображаемым, в случае с разделителем) лишь тогда, когда все выделенные изображения подходят под его список масок — если хотя бы один из файлов не подходит, инструмент скрывается. Это позволяет настраивать список инструментов под обработку лишь файлов определённого типа.

Правила составления масок описаны здесь.

NB: Пустая маска инструмента (не содержащая символов) означает применимость его к любому файлу.
{[%hh_h2:Использование инструмента]}

Инструменты в PhoA отображаются в различных меню. Вы можете настроить сами, где именно будет создан пункт для вызова действия, выполняемого инструментом:

  • В меню Сервис: команда отображается в подменю Сервис главного меню.
  • В контекстном меню окна групп изображений: команда появляется в меню дерева групп, вызываемом нажатием правой кнопки мыши.
  • В контекстном меню окна эскизов: команда появляется в меню окна просмотра эскизов, вызываемом нажатием правой кнопки мыши.
  • В контекстном меню режима просмотра: команда появляется в контекстном меню, вызываемом нажатием правой кнопки мыши в режиме просмотра.

Если в момент вызова команды инструмента было активно окно групп изображений, инструмент применяется ко всем изображениям группы. Если же было активно окно эскизов, то инструмент применяется к выделенным изображениям.
