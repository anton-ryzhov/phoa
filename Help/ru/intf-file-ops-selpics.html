{[%hh_hdr_ru:Мастер операций с файлами изображений :: Выбор изображений]}

Эта страница Мастера служит для выбора изображений и применима для следующих операций:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку
  • Удаление ненужных изображений вместе с файлами
  • Перестройка эскизов изображений
  • Восстановление ссылок на файлы изображений
{[%hh_h2:Элементы интерфейса]}
Выбранные изображения Выбранная Вами операция будет выполняться с изображениями, выбранными в окне просмотра эскизов. В случаях, когда операции требуется группа изображений, используется непосредственно выбранная в дереве группа фотоальбома или представления.
Этот режим выбирается автоматически, если вызов Мастера производится в момент, когда окно просмотра эскизов активно, и там имеются выбранные изображения.
Все изображения фотоальбома/представления Операция будет использовать все изображения и все группы фотоальбома (при просмотре групп изображений) или представления (когда активно представление).
Изображения в выбранных группах Вы выбираете, к каким группам и изображениям будет применяться операция, путём простановки отметки в приведённом дереве групп. Отметку можно установить только у групп, которые содержат изображения.
Этот режим выбирается автоматически, если вызов Мастера производится в момент, когда активно дерево групп; текущая выбранная группа сразу становится отмеченной (если в ней есть изображения).
Дополнительные опции доступны через контекстное меню, вызываемое нажатием правой кнопки мыши или комбинацией клавиш Shift+F10
Отметить все группы (клавиши Ctrl+A) — проставляет отметку у всех групп, имеющих изображения.
Снять отметку у всех групп (клавиши Ctrl+D) — удаляет отметку у всех отмеченных групп.
Инвертировать отметку у групп (клавиши Ctrl+I) — меняет состояние отметки у всех групп на противоположное.

Строка информации в нижней части страницы отображает, сколько именно групп и изображений отобрано для операции; также отображается суммарный размер, занимаемый файлами этих изображений.

NB:
  • Если опции копирования/перемещения позволяют создавать дубликаты файлов, суммарный их объём может оказаться больше указанного.
  • Занимаемое файлом на диске место, как правило, больше его размера — это следует учитывать при подготовке набора файлов для записи на сменный носитель.
