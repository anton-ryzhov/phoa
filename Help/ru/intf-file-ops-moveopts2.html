Эта страница Мастера служит для задания параметров следующих операций с файлами:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку

Если исходный файл не существует

Имеет смысл только для операции перемещения файлов и определяет поведение программы, если в процессе перемещения обнаружено отсутствие исходного файла:

  • Считать ошибкой — протоколирует ошибку и пропускает изображение.
  • Обновить ссылку только если файл назначения существует — если файл, который определён как файл назначения, существует — в изображении будет установлена ссылка на него, иначе изображение будет пропущено. Этот режим полезен в случае, если Вы переместили файлы, не пользуясь средствами программы, и хотите восстановить правильные ссылки — для этого укажите папку, где находятся новые файлы (необходимо учесть также прежнюю и новую структуру каталогов).
  • Обновить ссылку в любом случае — ссылка на файл изображения обновляется всегда, независимо от наличия исходного и целевого файлов.

Удалять исходные файлы

Имеет смысл только для операции перемещения файлов. Если опция включена, после успешного копирования и исправления ссылки файл будет удалён.

Удалять в Корзину

Имеет смысл только для операции перемещения файлов.

  • Если эта опция выключена, файл удалится окончательно и бесповоротно.
  • Если опция включена, файл будет перемещён в Корзину (если это возможно на данном компьютере).

Если файл назначения уже существует

Задаёт режим перезаписывания существующих файлов:

  • Пропускать текущее изображение — если файл назначения существует, программа посчитает это ошибкой, занесёт её в протокол и оставит прежний файл на месте.
  • Выдать запрос на перезапись файла — при обнаружении существующего файла программа выдаст запрос на перезапись файла. В случае отказа это изображение также будет занесено в протокол как сбойное.
  • Всегда перезаписывать файл — существующий файл будет перезаписан новым без каких-либо предупреждений.
