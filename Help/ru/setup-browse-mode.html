Деревья групп изображений
Всплывающие описания Определяет использование всплывающих подсказок (tooltips) в деревьях, отображающих группы изображений:
  • Не отображать — подсказки отключены.
  • Отображать, если строка не помещается в окне — подсвечивает подсказкой пункты, не помещающиеся в окне — так, чтобы их было видно целиком.
  • Отображать с выбранными свойствами — используются всплывающие подсказки, в которых отображаются выбранные свойства групп (см. ниже). Кроме того, в этом режиме для узла фотоальбома выводится в виде подсказки его описание.
Отображать во всплывающих описаниях Список свойств, из которого Вы можете выбрать, какие свойства групп отображать во всплывающих подсказках. Имеет смысл только когда для подсказок выбран режим Отображать с выбранными свойствами.
Окно просмотра эскизов
Цвет фона Задаёт цвет фона окна эскизов.
Цвет фона эскиза Задаёт цвет фона отдельных эскизов.
Цвет шрифта эскиза Задаёт цвет шрифта надписей эскизов.
Drag'n'Drop Разрешает или запрещает перетаскивание изображений из окна эскизов.
Показывать всплывающие описания эскизов Эта функция разрешает отображать всплывающие подсказки (tooltips), если остановить мышь над эскизом. Эти подсказки отображают свойства изображения (см. ниже). Вообще-то полезная функция, но некоторых раздражает.
Отображать во всплывающих описаниях Список свойств, из которого Вы можете выбрать, какие свойства изображений отображать во всплывающих описаниях эскизов.
Данные, отображаемые на эскизах С помощью этих опций Вы можете выбрать отображать ли и какие именно из свойств изображений в каждом из четырёх углов эскиза.
Стиль рамки фона эскиза Определяет стиль рамки прямоугольника, на котором отображаются изображение и описание эскиза в окне эскизов.
Пользовательский цвет рамки фона эскиза Если опция Стиль рамки фона эскиза установлена в Цвет, заданный пользователем, то цвет, указанный в данной настройке, используется для отрисовки границы прямоугольника эскиза.
Отображать тень эскиза Если включена, под изображением эскиза рисуется тень с параметрами, указанными ниже.
Радиус размывки тени Радиус "пятна", используемого для отрисовки тени, в пикселах.
Горизонтальное/вертикальное смещение тени Смещение тени в пикселах относительно изображения эскиза.
Цвет тени Цвет, используемый для отрисовки тени эскиза. По умолчанию — чёрный.
Непрозрачность тени Значение в диапазоне 0...255, задающее уровень "непрозрачности" тени эскиза.
Метод ресэмплинга эскизов Определяет интерполяционный фильтр, используемый при ресэмплинге (уменьшении) изображений для получения эскизов.
Количество операций в буфере отмены Задаёт размер буфера, используемого для хранения данных для отмены операций. Вы можете отменить не более указанного количества операций подряд (при заполнении буфера удаляются данные о самых ранних операциях). Большой размер буфера может потребовать значительного объёма памяти для хранения (особенно объёмными являются операции удаления и редактирования изображений).
