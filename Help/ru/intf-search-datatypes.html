{[%hh_hdr_ru:Поиск изображений :: Типы данных изображений]}

Согласно таблице данных изображения, PhoA использует следующие типы данных в свойствах изображений:
Тип данных Описание и критерии
Целое или натуральное число Данные представляют собой обычное целое число, которое можно проверить на равенство, неравенство и т. д.
Строка Любые символьные данные. Поиск по строке или подстроке в любых вариантах. Поиск регистронезависим.
Пиксельный формат Не является поисковым типом.
Дата Данные проверяются по критериям, аналогичным тем, что используются у чисел.
Время Данные проверяются по критериям, аналогичным тем, что используются у чисел.
Размеры (Ширина х Высота) Не является поисковым типом.
Список строк Это список ключевых слов. Слова можно вводить вручную или нажать кнопку "..." в поле ввода для выбора ключевых слов из списка. Критерии включают условия присутствия и отсутствия заданных (или всех) слов.
Поворот изображения Не является поисковым типом.
Отражения изображения Не является поисковым типом.
