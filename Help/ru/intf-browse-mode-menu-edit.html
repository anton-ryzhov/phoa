{[%hh_hdr_ru:Режим обзора :: Меню "Правка"]}

В этом меню собраны команды для редактирования фотоальбома, а также для перехода в режим просмотра.
Команда Описание
{[%hh_ii:Undo]} Отменить имя_операции Отменяет изменения, сделанные вами при выполнении указанной (последней) операции.
{[%hh_ii:UndoHistory]} История изменений Предоставляет доступ к списку сделанных изменений, с помощью которого вы можете отменить несколько сделанных операций сразу.
{[%hh_ii:Cut]} Вырезать Удаляет выделенные изображения в буфер обмена (вы можете впоследствии вставить их в другую группу).
{[%hh_ii:Copy]} Копировать Копирует выделенные изображения в буфер обмена.
{[%hh_ii:Paste]} Вставить Позволяет добавить изображения из буфера обмена в текущую группу, если они там есть. Естественно, добавляются только изображения, новые для текущей группы — совпадающие изображения отсеиваются. Если это указано в настройках программы, выводится уведомление о количестве реально вставленных изображений.
{[%hh_ii:NewGroup]} Добавить группу Добавление новой группы. Новая папка группы создаётся в текущей выделенной папке. Если, соответственно, выделен узел фотоальбома, то группа будет создана в его корневом каталоге. По умолчанию группе даётся имя "Новая группа", которое нужно изменить и нажать клавишу Enter.
{[%hh_ii:NewPic]} Добавить изображения... Запускает Мастер добавления изображений, позволяющий произвести добавление изображений (т. е. файлов) в фотоальбом.
{[%hh_ii:Delete]} Удалить
  • Когда активно дерево групп — удаление группы.
  • Когда активно окно эскизов — удаление выделенных в нём изображений.
{[%hh_ii:Edit]} Изменить...
  • Когда выделен узел фотоальбома (корневой узел дерева при отображении дерева групп), выбором этого пункта для него можно ввести описание и изменить параметры эскизов в окне свойств фотоальбома.
  • Когда выделена одна из групп фотоальбома, выбором этого пункта её можно переименовать или ввести описание для неё. (также переименовать группу можно нажав один раз на выделенном наименовании группы, или с помощью клавиши F2. Для отмены редактирования нужно нажать клавишу Esc, для сохранения нового имени группы — клавишу Enter).
  • Когда выделен узел представления (корневой узел дерева при отображении представления), с помощью этого пункта вы можете изменить его свойства.
{[%hh_ii:Sort]} Сортировать изображения... Предоставляет доступ к окну сортировки изображений для упорядочивания изображений группы или всех групп фотоальбома.
{[%hh_ii:SelectAll]} Выделить все Выделяет все изображения текущей группы.
{[%hh_ii:SelectNone]} Снять выделение Убирает выделение со всех изображений группы.
{[%hh_ii:ViewMode]} Просмотр Служит для перехода в режим просмотра. Также в режим просмотра можно перейти, сделав двойной щелчок мышью на изображении, или нажав Enter, когда активно окно эскизов.
{[%hh_ii:SlideShow]} Просмотр слайдов Запускает режим просмотра текущего изображения и включает режим показа слайдов.
