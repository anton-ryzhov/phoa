{[%hh_hdr_ru:Мастер операций с файлами изображений :: Опции создания CD/DVD]}

Эта страница Мастера служит для задания дополнительных параметров создания CD, DVD и прочих сменных носителей при следующих операциях с файлами:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку
{[%hh_h2:Элементы интерфейса]}
Создать файл фотоальбома с выбранными изображениями Если включено, программа создаст новый фотоальбом на основе выбранных групп и изображений, и поместит его в файл, указанный в следующем поле.
Имя файла фотоальбома Указывает имя файла, в который сохранять фотоальбом (с расширением .phoa). По умолчанию подставляется имя файла текущего фотоальбома.
Описание фотоальбома Позволяет задать описание для вновь создаваемого фотоальбома. По умолчанию подставляется описание текущего фотоальбома.
Также включить в фотоальбом существующие представления Если включено, в созданный фотоальбом будут добавлены все представления текущего фотоальбома.
Скопировать саму программу в папку назначения Если включено, исполняемый файл программы (phoa.exe) также будет скопирован в папку назначения. Поскольку, как известно, программа не требует инсталляции для запуска на компьютере, она может быть запущена с любого носителя, в том числе и сменного.
Скопировать текущие настройки программы в phoa.ini Если включено, в процессе выполнения копирования/перемещения текущие настройки программы будут сохранены в файле phoa.ini в папке назначения. Это даёт возможность запускать программу со сменного носителя с сохранением всех текущих настроек на новом компьютере (если опция автозагрузки параметров из phoa.ini не отключена в настройках на этом компьютере.)
Создать файл autorun.inf Эта опция даёт возможность автоматически сгенерировать файл autorun.inf, расположенный в папке назначения. В этом случае вставка CD-ROM в компьютер будет автоматически запускать PhoA, а программа после запуска будет загружать указанный выше фотоальбом.
NB: Автозапуск сработает только в том случае, если эта функция на компьютере не отключена.
Установить метку носителя Задаёт название носителя, отображаемое в папке Мой компьютер. По умолчанию сюда подставляется Фотоальбом или наименование текущего представления.
NB:
  • Это поле является необязательным, и его можно оставить пустым. В этом случае в папке Мой компьютер будет отображаться метка диска.
  • Метка носителя отображается только в случае, когда функция автозапуска на компьютере не отключена.

ВНИМАНИЕ: содержимое папки, указанной как папка назначения, должно после записи располагаться в корневом каталоге диска! {[%hh_seealso_ru]} Мастер операций с файлами изображений
