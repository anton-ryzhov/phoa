Мастер операций с файлами изображений :: Опции копирования/перемещения, страница 1

Эта страница Мастера служит для задания параметров следующих операций с файлами:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку

Режим размещения файлов

Эти опции определяют способ расположения файлов в выбранной папке назначения.

Расположить все файлы в единственной папке

В этом режиме все файлы помещаются в папке, указанной как папка назначения.

NB: Следует принять во внимание, что имена файлов могут совпадать. Это может создать конфликт при размещении всех файлов в единственной папке.

Сохранять исходную структуру папок

В таком варианте программа располагает файлы, копируя структуру папок, в которых они находятся. Пример см. ниже.

Использовать путь относительно папки

PhoA автоматически определяет общую для всех выбранных изображений часть пути и помещает возможные допустимые варианты общего пути в данный список.

Пример размещения файлов с сохранением исходной структуры папок

Допустим, мы выбрали для копирования/перемещения следующие изображения:

  • C:\Мои документы\Фотоальбом\Весна 1812\pic001.jpg
  • C:\Мои документы\Фотоальбом\Лето 1812\pic002.jpg
  • C:\Мои документы\Фотоальбом\Осень 1812\pic003.jpg
  • C:\Мои документы\Фотоальбом\Зима 1812\pic004.jpg
  • C:\Мои документы\Фотоальбом\Зима 1812\Москва\pic005.jpg

Программа автоматически определяет, что самый длинный общий путь у этих изображений - это C:\Мои документы\Фотоальбом, и подставляет в список возможных базовых папок его и остальные допустимые варианты: C:\Мои документы и C:\; последним добавляется вариант (нет): при выборе его буква исходного диска также превратится в папку внутри папки назначения.

Предположим теперь, что мы выбрали в качестве базовой папки C:\Мои документы, а в качестве папки назначения, скажем, D:\CD-ROM\Disk1. После выполнения операции мы получим следующие файлы:

  • D:\CD-ROM\Disk1\Фотоальбом\Весна 1812\pic001.jpg
  • D:\CD-ROM\Disk1\Фотоальбом\Лето 1812\pic002.jpg
  • D:\CD-ROM\Disk1\Фотоальбом\Осень 1812\pic003.jpg
  • D:\CD-ROM\Disk1\Фотоальбом\Зима 1812\pic004.jpg
  • D:\CD-ROM\Disk1\Фотоальбом\Зима 1812\Москва\pic005.jpg

Программа возьмёт D:\CD-ROM\Disk1 в качестве отправной точки, и каждый файл изображения будет иметь в своём пути папку Фотоальбом, поскольку путь берётся относительно папки C:\Мои документы.

Помещать файлы в папки, повторяя структуру групп изображений

В этом режиме PhoA будет раскладывать файлы по папкам, копируя существующую иерархию групп в фотоальбоме или представлении (смотря что в данный момент отображается).

Использовать путь относительно группы

В этом списке Вы можете указать, начиная с какой группы программа копирует структуру папок. По умолчанию подставляется самая дальняя от корневого элемента (фотоальбома или представления) группа, являющаяся родительской для всех выбранных групп изображений (так, если папка фотоальбома тоже входит в число выбранных, или выбрано копирование/перемещение всего фотоальбома, в этом списке будет единственный элемент - Фотоальбом). Также в список заносятся пути ко всем родительским группам этой группы, включая и окончательного родителя всех групп - Фотоальбом или текущее представление.

Разрешать несколько копий файла

Поскольку в PhoA одно изображение может входить в несколько групп, при копировании структуры групп в виде папок может потребоваться положить несколько копий одного файла в разные папки. Если данный переключатель включен, программа так и поступит, в противном случае файл будет скопирован только единожды (в первую папку по направлению обхода).

Заменять недопустимые символы символом

Задаёт символ, на который заменять символы, недопустимые в именах папок (это следующие символы: \ / : * ? " < > |). Например, если этот символ установлен в _ (по умолчанию), а группа называется Друзья/Враги, то для неё будет создана папка Друзья_Враги.
