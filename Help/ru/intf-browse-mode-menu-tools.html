Здесь расположены дополнительные инструменты для работы с фотоальбомами.
Команда Описание
{[%hh_ii:Props]} Настройки программы... Отображает окно настроек программы.
{[%hh_ii:Stats]} Статистика... Подсчитывает и выводит различные сводные данные по фотоальбому.
{[%hh_ii:PicOps]} Операции с изображениями... Позволяет выполнять перемещение, копирование и другие операции с изображениями.
{[%hh_ii:FileOps]} Операции с файлами изображений... Запускает Мастер операций с файлами изображений, позволяющий скопировать, переместить, удалить файлы изображений, перестроить эскизы или восстановить повреждённые ссылки на файлы изображений.
{[%hh_ii:Search]} Поиск... Открывает окно поиска изображений. Более подробно о поиске рассказано здесь.
