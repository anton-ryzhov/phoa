{[%hh_hdr_ru:Окно операций с изображениями]}

Функция позволяет произвести несколько операций между выделенными изображениями и одной из групп (кроме текущей) фотоальбома. Эти операции включают:

  • Перемещение выделенных изображений в выбранную группу (аналог простого перетаскивания выделенных изображений мышью в эту группу);
  • Добавление (копирование) выделенных изображений к выбранной группе (аналог перетаскивания выделенных изображений мышью в эту группу с нажатой клавишей Ctrl);
  • Удаление выделенных изображений из выбранной группы (группа будет содержать все прежние изображения, кроме копий выделенных изображений);
  • Пересечение выделенных изображений с изображениями выбранной группы (группа будет содержать только те изображения, которые идентичны выделенным).

Это позволяет выполнять различные задачи при упорядочивании содержимого фотоальбома (например, можно убрать повторяющиеся изображения из другой группы или скопировать часть изображений из одной группы в другую).

Эта операция поддерживает откат
