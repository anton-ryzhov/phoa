Справочная система PhoA

Описание команд меню в режиме обзора

Меню "Файл"

В этом меню собраны команды работы с файлами фотоальбомов:

  • Создание нового фотоальбома происходит автоматически при запуске программы или при выборе пункта меню Файл | Новый. При этом удаляются все группы, изображения и представления, а параметры фотоальбома приводятся к заданным по умолчанию.
  • Чтобы открыть ранее сохранённый файл фотоальбома, нужно выбрать пункт меню Файл | Открыть.... Это вызовет диалог открытия файла фотоальбома (с расширением .phoa).
  • Для сохранения файла фотоальбома с текущим именем выбирается пункт Файл | Сохранить.
  • Для сохранения в другой файл - пункт Файл | Сохранить как....
  • Пункт Файл | Сохранить настройки в файл... отображает диалог для сохранения текущих настроек программы в файл с расширением .ini, подробнее.
  • Пункт Файл | Загрузить настройки из файла... отображает диалог для загрузки настроек программы из файла с расширением .ini, подробнее.
  • Пункт меню Файл | Выход запрашивает подтверждение в случае несохранённого фотоальбома (или если запрос подтверждения при выходе задан в настройках подтверждений) и завершает работу с программой.
  • Далее следуют пункты меню последних открывавшихся файлов (их количество настраивается с помощью общих настроек программы).

Меню "Правка"

Здесь собраны команды для редактирования фотоальбома:

  • Пункт меню Правка | Отменить имя_операции отменяет изменения, сделанные Вами при выполнении указанной (последней) операции.
  • Подменю Правка | История изменений предоставляет доступ к списку сделанных изменений, с помощью которого Вы можете отменить несколько сделанных операций сразу.
  • Пункт меню Правка | Вырезать удаляет выделенные изображения в буфер обмена (вы можете впоследствии вставить их в другую группу).
  • Пункт меню Правка | Копировать, аналогично, копирует выделенные изображения в буфер обмена.
  • Пункт Правка | Вставить позволяет добавить изображения из буфера обмена в текущую группу, если они там есть. Естественно, добавляются только изображения, новые для текущей группы - совпадающие изображения отсеиваются. Если это указано в настройках программы, выводится уведомление о количестве реально вставленных изображений.
  • На вершине иерархии папок находится фотоальбом - он является корневым узлом дерева. Для него можно ввести описание и изменить параметры эскизов в окне свойств фотоальбома - делается это выбором пункта меню Правка | Изменить..., когда узел фотоальбома выделен.
  • Группу можно переименовать, выбрав пункт меню Правка | Изменить..., когда группа выделена в дереве, или нажав один раз на выделенном наименовании группы, или с помощью клавиши F2. После этого для отмены редактирования нужно нажать клавишу Esc, для сохранения нового имени группы - клавишу Enter.
  • Добавление новой группы осуществляется выбором пункта меню Правка | Добавить группу. При этом в дереве создаётся новая папка группы - в текущей выделенной папке. Если выделен узел фотоальбома, то группа будет создана в корневом каталоге альбома. По умолчанию группе даётся имя "Новая группа", которое нужно изменить и нажать клавишу Enter.
  • Удаление группы производится выбором пункта меню Правка | Удалить, когда активно дерево папок.
  • Чтобы добавить изображения в группу, необходимо выбрать пункт меню Правка | Добавить изображение..., при этом откроется окно Мастера добавления изображений.
  • Для упорядочивания изображений пункт меню Правка | Сортировать изображения... предоставляет доступ к окну сортировки изображений.
  • Пункт меню Правка | Выделить все выделяет все изображения текущей группы.
  • Пункт Правка | Снять выделение наоборот, убирает выделение со всех изображений группы.
  • Для перехода в режим просмотра нужно выбрать пункт меню Правка | Просмотр или сделать двойной щелчок мышью на изображении.

Меню "Вид"

Это меню контролирует отображение элементов программы и представления:

  • Показ панели инструментов (Вид | Панель инструментов) и
  • Показ строки состояния (Вид | Строка состояния). Программа запоминает состояние этих пунктов и восстанавливает при следующем запуске.
  • Далее следуют пункты для работы с представлениями фотоальбома:
    • Пункт Группы изображений служит для перехода к основному режиму просмотра дерева папок, когда в левом окне программы отображаются группы фотоальбома (режим по умолчанию).
    • Пункт Новое представление... открывает окно свойств для создания нового представления.
    • Пункт Удалить представление удаляет текущее представление (если в данный момент активно именно представление).
    • Пункт Изменить представление... открывает окно свойств для редактирования текущего представления (если в данный момент активно именно представление).
    • Пункт Поместить в группу... вызывает диалоговое окно выбора группы фотоальбома, позволяющее создать копию папок представления в группах фотоальбома.

Меню "Сервис"

Здесь расположены дополнительные инструменты для работы с фотоальбомами:

  • Пункт Сервис | Настройки программы... отображает окно настроек режима обзора PhoA.
  • Пункт Сервис | Статистика... подсчитывает и выводит различные сводные данные по фотоальбому.
  • Пункт Сервис | Операции с изображениями... позволяет выполнять перемещение, копирование и другие операции с изображениями.
  • Пункт Сервис | Операции с файлами изображений... запускает Мастер операций с файлами изображений, позволяющий скопировать, переместить, удалить файлы изображений, перестроить эскизы или восстановить повреждённые ссылки на файлы изображений.
  • Одной из самых ценных функций PhoA является поиск, окно которого вызывается выбором пункта Сервис | Поиск.... Более подробно о поиске рассказано здесь.

Меню "Справка"

Меню состоит из трёх пунктов:

  • Пункт Справка | О программе... вызывает окно с информацией о версии программы, об авторских правах и координатах для поиска обновлённых версий.
  • Пункт Справка | Содержание справки запускает Справочную систему PhoA, которую Вы в данный момент и наблюдаете.
  • Пункт Справка | FAQ отображает раздел справки FAQ (часто задаваемые вопросы).
  • Пункт Справка | Web-сайт программы запускает Интернет-браузер по умолчанию и открывает в нём домашнюю страницу PhoA.

Непосредственно слева от узла фотоальбома в дереве папок расположена кнопка, вызывающая меню представлений.
