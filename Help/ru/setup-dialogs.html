Отображать заставку при запуске

Если включено, при запуске программы отображается окошко заставки.

Анимированное скрытие заставки при запуске

Если включено, заставка, отображаемая при запуске, будет плавно "растворяться" после того, как программа вошла в ждущий режим (только на Windows 2000 и более поздних).

Анимированное скрытие диалога «О программе»

Если включено, диалог О программе будет плавно "растворяться" при закрытии (только на Windows 2000 и более поздних).

Подтверждения

Перед удалением группы

Если включено, перед удалением группы выдаётся соответствующий запрос.

Перед удалением изображений

Если включено, перед удалением изображений выдаётся соответствующий запрос.

Перед удалением представления

Если включено, перед удалением представления выдаётся соответствующий запрос.

Перед сохранением файла устаревшей версии

Если включено, то при сохранении файла более старой версии, чем текущая, выдаётся предупреждение о возможной потере данных.

Перед выходом (при сохранённом фотоальбоме)

Если включено, перед выходом из программы выдаётся запрос даже при неизменённом фотоальбоме (при изменённом фотоальбоме запрос выдаётся всегда).

Уведомления

После копирования изображений с помощью Drag'n'Drop

Если включено, после копирования изображений перетаскиванием выдаётся информация о том, сколько реально изображений было скопировано.

После перемещения изображений с помощью Drag'n'Drop

Если включено, после перемещения изображений перетаскиванием выдаётся информация о том, сколько реально изображений было перенесено.

После вставки изображений из буфера обмена

Если включено, после вставки изображений из буфера обмена выдаётся информация о том, сколько реально изображений было вставлено.

Мастер добавления изображений

Отображать скрытые файлы и папки

Контролирует отображение файлов и папок, имеющих атрибут Скрытый.

Пропускать страницу отметки файлов

Если включено, то после выбора файлов/папок в Мастере добавления изображений PhoA сразу начинает процесс добавления изображений (пропуская страницу отметки файлов).

Отображать протокол добавления только при наличии ошибок

Если включено, протокол добавления будет отображаться только в том случае, если при добавлении происходили сбои, в противном случае окно Мастера добавления закроется сразу по окончании процесса добавления изображений. Если выключено, протокол будет отображаться всегда по завершении процесса.

Автозаполнение даты изображения из:

Настраивает свойства, которые будут участвовать в автозаполнении даты изображения при добавлении.

Переписывать дату, если она уже указана

Заставляет программу производить автозаполнение даты изображения (переписывать дату), даже если дата изображения уже введена.

Автозаполнение времени изображения из:

Настраивает свойства, которые будут участвовать в автозаполнении времени изображения при добавлении.

Переписывать время, если оно уже указано

Заставляет программу производить автозаполнение времени изображения (переписывать время), даже если время изображения уже введено.

Окно свойств изображений

Сразу раскрывать свойства файлов

Если включено, свойства файлов на странице свойств раскрываются сразу полностью.

Сразу раскрывать метаданные изображений

Если включено, метаданные изображений на странице метаданных раскрываются сразу полностью.

Мастер операций с файлами изображений

Подтверждение для операции копирования файлов

Если включено, перед непосредственным запуском процесса копирования файлов программа запрашивает подтверждения.

Подтверждение для операции перемещения файлов

Если включено, перед непосредственным запуском процесса перемещения файлов программа запрашивает подтверждения.

Подтверждение для операции удаления файлов

Если включено, перед непосредственным запуском процесса удаления файлов программа запрашивает подтверждения.

Подтверждение для операции перестройки эскизов

Если включено, перед непосредственным запуском процесса перестройки эскизов программа запрашивает подтверждения.

Подтверждение для операции восстановления ссылок на файлы

Если включено, перед непосредственным запуском процесса восстановления ссылок на файлы программа запрашивает подтверждения.

Отображать протокол операции только при наличии ошибок

Если включено, протокол файловой операции будет отображаться только в том случае, если в процессе выполнения её происходили сбои, в противном случае окно Мастера операций с файлами закроется сразу по окончании операции. Если выключено, протокол будет отображаться всегда по завершении процесса.
