{[%hh_hdr_ru:Типичные задачи :: Сортировка изображений]}

Под сортировкой изображений в PhoA понимается упорядочивание их по каким-либо признакам.

NB: Каждая группа изображений имеет свой набор ссылок на изображения. Поэтому вполне могут существовать, например, две группы с совершенно одинаковым набором изображений, но упорядоченных по-разному.

Программа позволяет упорядочить изображения по любым доступным их свойствам. Наиболее естественным способом сортировки, например, для фотографий, является сортировка их в хронологической последовательности, т. е. по дате и времени.

Вы можете сортировать изображения либо в текущей группе, либо во всех группах фотоальбома сразу. Чтобы сделать это:

  1. Если вы хотите сортировать изображения только в одной группе, выделите эту группу в дереве.
  2. Выберите пункт меню Правка | Сортировать изображения: появится диалоговое окно сортировки.
  3. В переключателе Где сортировать изображения выберите подходящий режим.
  4. Задайте список сортировок. Кнопка со стрелкой вниз позволяет задать свойство для сортировки или удалить строку. Чтобы добавить новое свойство, задайте его в последней, незаполненной строке таблицы.
  5. Нажмите ОК.
NB: Точно таким же образом можно отсортировать результаты поиска изображений, но для этого надо выделить папку результатов в дереве групп и явно указать режим В текущей группе.
