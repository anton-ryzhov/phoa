{[%hh_hdr_ru:Окно свойств изображения :: Метаданные]}

Эта страница отображает метаданные, содержащиеся в файлах изображений. Метаданные выводятся для каждого файла изображения, вы можете просмотреть их, раскрыв имя файла.

Если файл не содержит метаданных, напротив его имени выводится соответствующая информация.

Нижняя панель, Описание, отображает дополнительную информацию о выбранном метатеге. Эта панель видима по умолчанию, но может быть скрыта, если нажать кнопку X на ней. Чтобы вновь отобразить эту панель, выберите соответствующий пункт из контекстного меню.

NB:
  • Вы можете указать в настройках диалогов сразу разворачивать метаданные для всего списка файлов.
  • Метаданные могут быть использованы для автозаполнения свойств изображений при добавлении их в фотоальбом. Указывается это также в настройках диалогов.
  • Названия метатегов не локализованы и приводятся на английском языке. Связано это с тем, что их шибко много :) и, кроме того, они постоянно добавляются. Кому нужно, тот разберётся.
  • Описания метатегов также не локализованы.
