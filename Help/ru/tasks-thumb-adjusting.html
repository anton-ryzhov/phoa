{[%hh_hdr_ru:Типичные задачи :: Изменение размеров или качества эскизов]}

Эскизы, создаваемые программой, по умолчанию имеют размер 150х150 пиксел и качество примерно 40% от максимума.

Эти значения сохраняются индивидально для каждого фотоальбома, и вы можете изменить их в его свойствах. {[%hh_h2:Изменение параметров эскизов]}

  1. Выделите в дереве групп корневой узел (узел Фотоальбом).
  2. Выберите пункт меню Правка | Изменить... — появится диалоговое окно свойств фотоальбома.
  3. Задайте требуемые параметры эскизов.
  4. Сохраните изменения, нажав ОК.

Однако само по себе изменение параметров эскизов влияет только на вновь добавляемые изображения и никак не действует на изображения, уже существующие в фотоальбоме. Чтобы эти эскизы имели такие же характеристики, их необходимо создать заново, т. е.  перестроить. {[%hh_h2:Перестройка эскизов]}

Перестройка эскизов в PhoA выполняется с помощью Мастера файловых операций.

  1. Запустите Мастер операций с файлами изображений. Делается это выбором пункта меню Сервис | Операции с файлами изображений....
  2. На первой странице Мастера выберите Перестроить эскизы и нажмите Далее.
  3. Поскольку изменение параметров эскизов, вероятнее всего, должно сказаться на всех эскизах фотоальбома, выберите Все изображения фотоальбома.
  4. После нажатия Далее программа начнёт перестройку эскизов для всех изображений.
