Я прочитал и понял всю документацию программы. Но как (в конце концов) мне создать фотоальбом???

Это проще, чем может показаться:

  • Вы можете создать несколько папок-групп для изображений. Делается это выбором пункта меню {[%hh_ii:NewGroup]} Правка | Добавить группу или нажатием Alt+Ins — при этом группа добавляется "внутри" текущей выделенной группы. Этот шаг можно пропустить и добавлять изображения непосредственно в фотоальбом.
  • Выделите папку в дереве, в которую будут добавляться изображения (группу или сам фотоальбом).
  • Выберите пункт меню {[%hh_ii:NewPic]} Правка | Добавить изображения.... Откроется Мастер добавления изображений.
  • На первой странице Мастера выберите папку, содержащую файлы изображений. Если в ней есть вложенные папки с изображениями, включите переключатель Загружать папки вместе с подпапками. Нажмите Далее >.
  • На странице отметки файлов можно выбрать, какие именно файлы из перечисленных требуется добавить в фотоальбом. По умолчанию отмечены все файлы. Нажмите Далее > — начнётся добавление изображений.
  • После добавления программа либо закроет окно Мастера и Вы увидите свои изображения, либо отобразит страницу с протоколом добавления (в этом случае нажмите Закрыть) — в зависимости от настроек.
  • Сохраните фотоальбом в файл. Всё!

Как изменить язык интерфейса?

Изначальный язык интерфейса определяется языком, выбранным при установке программы. Изменить его можно, выбрав пункт меню {[%hh_ii:Props]} Сервис | Настройки программы... -> Вкладка {[%hh_ii:Props]} Общие -> Язык интерфейса ({[%hh_ii:Props]} Tools | Program settings... -> Вкладка {[%hh_ii:Props]} General -> Interface language)

Как мне изменить размер эскизов, используемых в программе?

Каждый фотоальбом может иметь собственный размер эскизов. Эти данные сохраняются вместе с фотоальбомом; их можно изменить, выделив узел фотоальбома в дереве групп и выбрав пункт меню {[%hh_ii:Edit]}Правка | Изменить...

В начало

Как можно улучшить качество эскизов?

На качество эскизов влияют два фактора:

  • Используемый метод ресэмплинга. Метод Nearest Neighbor является самым быстрым, но изображения, уменьшенные с его помощью, выглядят "мохнатыми". Наиболее плавные контуры даёт B-spline, потом Linear. Наиболее чёткие линии на уменьшенных изображениях обеспечивает Lanczos. Изменить способ ресэмплинга можно в настройках программы ({[%hh_ii:Props]} Сервис | Настройки программы... -> {[%hh_ii:Folder]} Режим обзора -> Метод ресэмплинга эскизов).
  • Уровень сжатия. Эскизы хранятся в формате JPEG; этот формат позволяет сжимать изображения ценой некоторой потери качества. Соотношение "размер-качество" можно задавать для каждого фотоальбома индивидуально. Изменить уровень сжатия (качество эскиза) можно, выделив узел фотоальбома в дереве групп и выбрав пункт меню {[%hh_ii:Edit]} Правка | Изменить...

Могу ли я записать созданный фотоальбом на CD-ROM вместе с программой — так, чтобы его можно было бы просмотреть на любом компьютере?

Да, программа PhoA специально была спроектирована так, чтобы фотоальбомы можно было записать на сменный носитель вместе с самим средством просмотра:

  • В файле фотоальбома всегда по возможности сохряняются относительные ссылки на файлы изображений (относительно файла фотоальбома). Абсолютные ссылки сохраняются только для тех изображений, файлы которых находятся на другом диске, чем файл фотоальбома, или на сетевом сервере. Это даёт возможность записать на диск изображения и фотоальбом, сохранив неизменным их взаимное расположение. Например:
    • До записи: файл фотоальбома: C:\Мои документы\Фотоальбом.phoa; базовый каталог файлов изображений: C:\Мои документы\Изображения
    • После записи (на CD-ROM): файл фотоальбома: D:\Фотоальбом.phoa; базовый каталог файлов изображений: D:\Изображения
  • Программа PhoA не требует для своего запуска никаких дополнительных файлов, кроме собственно исполняемого файла phoa.exe. Его можно разместить в любом каталоге компакт-диска.
  • Чтобы при запуске программа отображала заданный фотоальбом, нужно путь к его файлу указать в командной строке, например: phoa.exe Фотоальбом.phoa
  • Наиболее простым и эффективным способом подготовки файлов для записи на сменный носитель является использование операции копирования файлов.

Как должен выглядеть файл autorun.inf, чтобы фотоальбом на CD-ROM запускался автоматически?

Простейший вариант файла autorun.inf (создаётся любым текстовым редактором и при записи помещается в корневой каталог CD-ROM-диска): 

[autorun]
open=phoa.exe Фотоальбом.phoa
icon=phoa.exe,1
label=Мой фотоальбом
Этот пример подразумевает, что исполняемый файл программы (phoa.exe) и файл фотоальбома (Фотоальбом.phoa) находятся в корневом каталоге диска (так же, как и сам autorun.inf).
Строка icon= указывает файл, содержащий значок, отображаемый в "Моём компьютере", а также индекс значка в этом файле (значки нумеруются с нуля). Для файла phoa.exe:
   0 — значок самой программы;
   1 — значок файла фотоальбома.
Строка label= задаёт метку устройства, отображаемую в "Моём компьютере". За более подробной информацией обращайтесь к документации MSDN.
NB: Можно воспользоваться функцией создания файла autorun.inf Операции копирования файлов.

Я хочу, чтобы при запуске программы автоматически открывался мой фотоальбом. Как?

Для этого Вы должны запускать не программу, а непосредственно сам файл фотоальбома. Например, создайте на рабочем столе ярлык к своему фотоальбому C:\Мои документы\Фотоальбом.phoa, и при запуске его сначала будет загружаться PhoA, потом программа загрузит этот фотоальбом.

В начало

Можно ли изменить путь к изображениям фотоальбома?

Во-первых, вы можете перемещать файлы изображений вместе с файлом фотоальбома, сохранив неизменным их взаимное расположение (см. запись на CD).
Во-вторых, вы можете изменять расположение файлов с помощью операции перемещения файлов.
И в-третьих, можно сменить файл изображения непосредственно, используя кнопку Изменить файл... на странице Свойства файла окна свойств изображения.
