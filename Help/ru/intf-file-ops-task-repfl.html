Эта операция позволяет восстановить испорченные по разным причинам ссылки на файлы изображений путём поиска соответствующих файлов в заданной папке.

Чтобы выполнить эту операцию, необходимо запустить Мастер операций с файлами изображений и на странице выбора задачи выбрать Восстановить ссылки на файлы изображений.
