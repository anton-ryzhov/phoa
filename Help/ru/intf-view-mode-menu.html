NB:
  • Большинство команд меню также доступно из контекстного меню режима просмотра, вызываемого нажатием правой кнопки мыши.
  • Нажатие правой кнопки мыши при удерживаемой клавише Ctrl вызывает системное контекстное меню (меню Проводника) для текущего файла изображения.
  • Если главное меню не отображается, чтобы показать его, выберите пункт контекстного меню Вид | Главное меню.
