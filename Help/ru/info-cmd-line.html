Справочная система PhoA

Параметры командной строки

PhoA поддерживает задание ряда параметров в командной строке. Общий формат командной строки программы таков:

phoa.exe [ключ1 [значение_ключа1] [ключ2 [значение_ключа2]] ... ] [файл_фотоальбома],

где:

  • ключN - символ, перед которым стоит знак - ("минус"). Список допустимых ключей см. ниже.
  • значение_ключаN - строка символов, определяющая значение ключа. Некоторые ключи не допускают указания значения, некоторые его обязательно требуют. Остальные, соответственно, допускают необязательное значение.
  • файл_фотоальбома - необязательный файл фотоальбома, загрузка которого произойдёт при запуске программы.

Допустимые ключи

-w имя_представления - активизировать представление имя_представления. Если данный ключ не указан, после запуска программа по умолчанию отображает группы изображений. Поиск представления по имени осуществляется без учёта регистра.
Пример: -w "По дате и месту"

-g путь_группы - выбрать в дереве группу, заданную параметром путь_группы. Путь группы представляет собой строку наподобие пути файла в файловой системе, с тем отличием, что для разделения каталогов используется символ / ("слэш"). В начале пути можно указывать, а можно и не указывать символ / - на результат это не влияет. Может быть указана как обычная группа, так и группа представления (т.е. в сочетании с ключом -w). Если данный ключ не указан, программа по умолчанию выделяет корневой элемент дерева групп. Поиск групп по имени осуществляется без учёта регистра.
Пример: -g "Фотографии по дате/2000 г./Январь/Утро"

-i ID_изображения - выделить в окне эскизов изображение с заданным ID. Если данный ключ не указан, программа по умолчанию выделяет самый первый эскиз в текущей группе.
Пример: -i 418

-v - после запуска перейти в режим просмотра. Переход в режим просмотра доступен только при наличии выделенного изображения.

-s - имеет смысл только в сочетании с -v. Если указан, программа после перехода в режим просмотра запустит показ слайдов.

-f [1|0] - имеет смысл только в сочетании с -v и задаёт полноэкранный или оконный режим. Если данный ключ не указан, то программа использует текущие настройки для выбора полноэкранного/оконного режима. Если ключ указан без значения или со значением 1 (единица), то принудительно устанавливается полноэкранный режим режима просмотра (только для текущей сессии). Если ключ указан со значением 0 (ноль), то принудительно устанавливается оконный режим режима просмотра (также только для текущей сессии).
Примеры:
    -f
    -f 0

NB:
  • Параметры в командной строке разделяются пробелами.
  • Если значение параметра или имя файла фотоальбома содержат пробелы, их необходимо заключать в двойные кавычки, как в вышеприведённых примерах.
  • Имя файла фотоальбома можно указывать в любом месте командной строки после имени программы. Необходимо обеспечить, однако, чтобы программа не восприняла его в качестве значения параметра: например, писать его в начале строки параметров или же после значения параметра (так как значение у параметра может быть только одно).
  • Ошибки в задании параметров командной строки отображаются при запуске программы.
