{[%hh_hdr_ru:Мастер добавления изображения :: Выбор файлов]}

Добавление изображений начинается с выбора файлов изображений и/или папок, содержащих файлы изображений.

Список файлов и папок

Вы должны выделить необходимые элементы в дереве файлов клавиатурой или мышью с помощью клавиш Shift и Ctrl.

Загружать папки вместе с подпапками

Если в дереве выделены папки, то при включенном переключателе выбираются файлы не только из них, но и файлы из всех вложенных папок.

Дополнительно

Отображает/скрывает панель Фильтра выбора файлов, позволяющего наложить дополнительные условия на выбираемые файлы (см. далее).

Наличие файлов в фотоальбоме

Позволяет отсеять файлы по их присутствию в фотоальбоме:

  • Не учитывать — фильтр отключен.
  • Только файлы, отсутствующие в фотоальбоме — будут добавлены только те файлы, которые не фигурируют в общем списке изображений альбома.
  • Только файлы, присутствующие в фотоальбоме — в группу добавятся только те файлы, которые уже есть в фотоальбоме.

Маски файлов

Фильтр, ограничивающий добавляемые файлы по их имени и/или расширению. Маска строится по стандартным правилам PhoA.

Время записи файла

Здесь можно указать границы даты и/или времени модификации файла. Таким образом, например, можно выбрать файлы, сохранённые ранее или позже заданной даты. Незаполненное поле означает неактивность фильтра.
