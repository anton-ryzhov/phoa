{[%hh_hdr_ru:Типичные задачи :: Исправление ссылки на файл изображения]}

Как известно, в группах фотоальбома PhoA содержатся ссылки на изображения, в то время как само изображение ссылается на соответствующий дисковый файл.

В сохранённом файле фотоальбома хранятся относительные ссылки на файлы изображений, если это возможно: это означает, что если вы перемещаете файл фотоальбома вместе с файлами изображений (сохраняя их взаимное расположение неизменным), то ссылки на файлы изображений в фотоальбоме останутся верными.

Для перемещения файлов изображений независимо от файла фотоальбома существует специальная операция перемещения файлов изображений.

Если же вы переместили файл изображения вне программы, ссылка на него становится неактуальной, «подвисшей», т.е. указывает в никуда.

Исправить это можно двумя способами:

  • Индивидуальный подход: вы можете указать новое местоположение файла.
  • Кардинальный способ: удалить все изображения, ссылающиеся на несуществующий файл. В этом случае вы, естественно, теряете все введёнными вами для этих изображений данные.
{[%hh_h2:Смена ссылки на файл изображения]}
  1. Выделите эскизы изображений, файлы которых нужно изменить.
  2. Выберите пункт меню Правка | Изменить...: откроется окно свойств изображений.
  3. Откройте страницу Свойства файла диалога свойств.
  4. Выберите один из файлов в списке и нажмите кнопку панели инструментов {[%hh_ii:Edit]} Сменить файл....
  5. В появившемся окне открытия файла укажите новое местоположение файла изображения и нажмите Открыть.
  6. Повторите пункты 4 и 5 для каждого из «подвисших» файлов.
  7. Сохраните изменения файлов нажатием ОК в окне свойств изображений.
{[%hh_h2:Операция удаления файлов изображений]}

Для удаления изображений, ссылающихся на несуществующий файл, сделайте следующее.

  1. Запустите Мастер операций с файлами изображений. Делается это выбором пункта меню Сервис | Операции с файлами изображений....
  2. На первой странице Мастера выберите Удалить изображения и файлы и нажмите Далее.
  3. Выберите Все изображения фотоальбома, а в Фильтре по существованию файла изображения выберите Только изображения, ссылающиеся на НЕсуществующий файл.

    ВНИМАНИЕ! Ошибка здесь может стоить вам всех ваших файлов, поэтому будьте внимательнее!

  4. Поскольку мы удаляем только изображения без соответствующего файла, опция удаления в Корзину не имеет никакого значения. После нажатия Далее программа начнёт удаление изображений.
