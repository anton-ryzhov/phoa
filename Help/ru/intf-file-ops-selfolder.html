Мастер операций с файлами изображений :: Выбор папки назначения

Эта страница Мастера служит для выбора папки назначения и применима для следующих операций:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку
  • Восстановление ссылок на файлы изображений

Вы должны выбрать папку в дереве или ввести путь к ней в строке ввода.

Создать папку

Эта кнопка создаёт папку внутри текущей выбранной в дереве, выделяет её и входит в режим редактирования её имени.

NB: Для операций копирования и перемещения файлов Вы можете указать имя новой, несуществующей папки - тогда она будет создана автоматически в процессе выполнения операции.
