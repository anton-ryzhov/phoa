Настройки программы :: Инструменты

Страница Инструменты Диалога настроек служит для определения набора доступных инструментов.

Порядок следования инструментов в списке определяет порядок их следования в меню.

Вы можете менять его, перетаскивая записи мышью, с помощью команд контекстного меню или клавишами Ctrl+Up, Ctrl+Down.

Чтобы отобразить меню для инструмента, нажмите кнопку слева от пункта, правую клавишу мыши или клавишу Пробел.

Удаление инструмента производится командой контекстного меню Удалить или клавишей Del.

Чтобы просмотреть или изменить свойства инструмента, выберите пункт контекстного меню Изменить..., нажмите комбинацию клавиш Alt+Enter или сделайте двойной щелчок мышью на пункте. При этом отобразится Диалог свойств инструмента.

Чтобы добавить новую запись инструмента, выберите команду Изменить... для последней, незаполненной записи списка.

Выбор столбцов

Вы можете выбрать, какие столбцы в таблице инструментов отображаются с помощью контекстного меню, вызываемого нажатием правой кнопки мыши на заголовке таблицы. Кроме того, Вы можете изменять порядок следования столбцов, перетаскивая их мышью.
