В этом меню собраны команды работы с файлами фотоальбомов.
Команда Описание
{[%hh_ii:New]} Новый Создание нового фотоальбома. При этом удаляются все группы, изображения и представления, а параметры фотоальбома приводятся к заданным по умолчанию.
{[%hh_ii:Open]} Открыть... Вызывает стандартный диалог открытия файла для выбора файла фотоальбома (с расширением .phoa).
{[%hh_ii:Save]} Сохранить Сохраняет фотоальбом в текущий открытый файл. Если это новый фотоальбом, при выборе этого пункта выполняется команда Сохранить как...
{[%hh_ii:SaveAs]} Сохранить как... Вызывает стандартый диалог сохранения файла — для сохранения текущего фотоальбома в файл с другим именем.
{[%hh_ii:SaveSettings]} Сохранить настройки в файл... Отображает диалог для сохранения текущих настроек программы в файл с расширением .ini, подробнее.
{[%hh_ii:LoadSettings]} Загрузить настройки из файла... Отображает диалог для загрузки настроек программы из файла с расширением .ini, подробнее.
{[%hh_ii:Exit]} Выход Запрашивает подтверждение в случае несохранённого фотоальбома (или если запрос подтверждения при выходе задан в настройках подтверждений) и завершает работу с программой.
Далее следуют пункты меню последних открывавшихся файлов (их количество настраивается с помощью общих настроек программы).
