{[%hh_hdr_ru:Мастер операций с файлами изображений :: Операция перестройки эскизов]}

Перестройка эскизов требуется прежде всего тогда, когда меняются параметры эскизов (определяемые в свойствах альбома): размеры, качество или метод ресэмплинга (последний задаётся в настройках режима обзора). В этом случае необходимо построить новые эскизы - в соответствии с новыми параметрами. Также эта операция может оказаться полезной, если изменились исходные изображения альбома.

Чтобы выполнить эту операцию, необходимо запустить Мастер операций с файлами изображений и на странице выбора задачи выбрать Перестроить эскизы.

Далее последовательность страниц такова:

  • Выбор изображений для операции
  • Процесс выполнения операции
  • Протокол операции
