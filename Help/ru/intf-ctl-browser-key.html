Клавиши Действие Описание
Alt+Enter Изменение Открывает окно свойств фотоальбома или изображения, или переходит в режим редактирования имени текущей группы (зависит от текущего выделения).
Alt+Ins Создание группы Создаёт новую группу в текущей группе, присваивает ей имя по умолчанию и входит в режим редактирования имени.
Ctrl+A Выделить все Выделяет все изображения текущей группы.
Ctrl+C, Ctrl+Ins Копировать Копирует выделенные изображения в буфер обмена.
Ctrl+D Снять выделение Убирает выделение со всех изображений текущей группы.
Ctrl+End Фокус в конец Фокусирует последнее изображение, не изменяя текущего выделения.
Ctrl+Enter Просмотр Переходит в режим просмотра текущего изображения.
Ctrl+F1 О программе Отображает информацию о версии, авторских правах и обновлении PhoA.
Ctrl+F2 Операции с изображениями Вызывает диалоговое окно операций с изображениями.
Ctrl+F3 Сортировка изображений Открывает окно сортировки изображений.
Ctrl+G Группы изображений Отобразить дерево групп фотоальбома (в случае, если активно представление).
Ctrl+Home Фокус в начало Фокусирует первое изображение, не изменяя текущего выделения.
Ctrl+N Создание нового фотоальбома Очищает всю информацию о фотоальбоме, группах и изображениях.
Ctrl+O Открытие файла фотоальбома Открывает окно для выбора файла фотоальбома, загружает фотоальбом для обработки или просмотра.
Ctrl+Page Down Фокус на страницу вниз Передвигает фокус изображения на экран вниз, не изменяя текущего выделения.
Ctrl+Page Up Фокус на страницу вверх Передвигает фокус изображения на экран вверх, не изменяя текущего выделения.
Ctrl+R Операции с файлами Открывает окно Мастера операций с файлами изображений.
Ctrl+S Сохранение файла фотоальбома Записывает текущий фотоальбом в файл на диске.
Ctrl+V, Shift+Ins Вставить Добавляет изображения из буфера обмена к текущей группе (если буфер обмена содержит изображения, помещённые туда программой PhoA).
Ctrl+X, Shift+Del Вырезать Удаляет выделенные изображения в буфер обмена.
Ctrl+Z, Alt+Backspace Отменить Отменяет последнее сделанное Вами изменение.
Ctrl+Пробел Переключить выделение эскиза Выделяет или снимает выделение с текущего эскиза.
Ctrl+Клавиши со стрелками Перемещение фокуса Передвигает фокус (пунктирное выделение) изображения, не изменяя текущего выделения.
Del Удаление Удаляет выбранную группу или выделенные изображения после выдачи запроса (зависит от текущего выделения).
End Последний элемент Перемещает выделение к последней группе или изображению (в зависимости от того, какое окно активно).
Enter (когда активно окно эскизов) Просмотр Переходит в режим просмотра текущего изображения.
F1 Содержание справки Запускает Справочную систему PhoA.
F2 (когда активно дерево групп) Переименовать Переходит в режим редактирования имени текущей группы фотоальбома или имени представления.
F3 Поиск Доступ к функции поиска изображений.
F4 Настройки программы Открывает окно настроек программы.
F8 Статистика Рассчитывает и отображает статистику фотоальбома.
F12 Сохранение фотоальбома в файл с другим именем Открывает окно для ввода имени нового файла фотоальбома, записывает фотоальбом в выбранный файл и делает этот файл текущим.
Home Первый элемент Перемещает выделение к первой группе или изображению (в зависимости от того, какое окно активно).
Ins Добавление изображения Открывает Мастер добавления изображений в текущую группу.
Page Down Следующая страница Перемещает выделение на один экран вниз.
Page Up Предыдущая страница Перемещает выделение на один экран вверх.
Shift+Alt+Del Удалить представление Удаляет текущее представление.
Shift+Alt+Enter Изменить представление Открыть окно свойств представления.
Shift+Alt+G Поместить представление в группу Открыть окно выбора группы для помещения в неё представления.
Shift+Alt+Ins Новое представление Открыть Окно добавления представления.
Shift+Alt+Клавиши со стрелками Выделение прямоугольной области изображений Выделяет прямоугольную область изображений (с предыдущего до следующего текущего).
Shift+Ctrl+Enter Просмотр слайдов Переходит в режим просмотра текущего изображения и включает показ слайдов.
Shift+Ctrl+O Загрузка настроек из ini-файла Отображает диалог для выбора файла с расширением .ini, из которого нужно загрузить настройки программы.
Shift+Ctrl+S Сохранение настроек в ini-файле Отображает диалог для выбора файла с расширением .ini, в который требуется записать текущие настройки программы.
Shift+End Выделение изображений до конца Выделяет подряд изображения с текущего до самого последнего.
Shift+Home Выделение изображений до начала Выделяет подряд изображения с текущего до самого первого.
Shift+Page Down Выделение изображений на страницу вниз Выделяет подряд изображения с текущего до изображения, находящегося на один экран ниже.
Shift+Page Up Выделение изображений на страницу вверх Выделяет подряд изображения с текущего до изображения, находящегося на один экран выше.
Shift+Клавиши со стрелками Выделение диапазона изображений Выделяет подряд несколько изображений (с предыдущего до следующего текущего).
Клавиши со стрелками Перемещение выделения Перемещает выделение по дереву групп или списку изображений (в зависимости от того, какое окно активно).
Пробел Выделить эскиз Выделяет текущий эскиз.
