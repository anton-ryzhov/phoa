{[%hh_hdr_ru:Окно свойств инструмента]}

Это окно служит для просмотра и редактирования свойств инструмента. Чтобы отобразить его, выберите команду Изменить... на странице Инструменты диалога настроек программы.

Маски файлов

Определяют набор файлов, к которым применим инструмент.

Вид

Задаёт вид инструмента.

Наименование

Определяет текст пункта меню, создаваемого для инструмента. Вы можете создать горячую клавишу для вызова инструмента, введя символ & перед одной из букв. Чтобы набрать символ & в тексте пункта, необходимо его продублировать.

Подсказка

Подсказка отображается в строке состояния. В случае, если в строке имеется символ | (вертикальная черта), то в строке состояния отображается текст после него; текст до этого символа отображается в виде всплывающей подсказки кнопки на панели инструментов.

Команда (приложение)

Имеет смысл только для пользовательского действия. Задаёт исполняемый файл, которому будут переданы файлы в качестве параметров командной строки. Чтобы выбрать файл, можно воспользоваться кнопкой Обзор....

Параметры

Задаёт дополнительные параметры командной строки, передаваемые приложению.

Окно

Определяет состояние главного окна указанного приложения после его запуска.

Папка

Задаёт рабочую папку приложения. Кнопка Обзор... позволяет выбрать папку.
