{[%hh_hdr_ru:Поиск изображений :: Простой поиск]}

Простой поиск позволяет наглядно задать условия выборки изображений на основе значений их свойств. {[%hh_h2:Элементы интерфейса]}
Сброс Стирает все введённые критерии.
Удалить Удаляет текущий выделенный критерий.
Преобразовать в выражение Составляет выражение для поиска по выражению на основе уже введённых критериев и помещает его в редактор выражения.
Список критериев поиска Вы вводите условия поиска в таблице критериев, которые определяются следующими данными:
  • Свойство изображения — один из поисковых элементов данных изображения, для которого вводится критерий.
  • Условие — одно из условий сравнения свойства, набор возможных условий определяется типом данных.
  • Значение — значение, с которым сравнивается свойство изображения.

Для того, чтобы добавить новый критерий, просто выберите необходимое свойство изображения, нажав кнопку меню в последней, незаполненной строке таблицы.

Чтобы передвигаться по ячейкам таблицы, используются клавиши со стрелками. Для раскрытия выпадающего списка существует комбинация клавиш Alt+Вниз.

Значения условий хранят историю вводимых вами значений.

Программа выбирает те изображения, которые соответствуют всем критериям поиска одновременно (то есть здесь действует условие И). {[%hh_seealso_ru]} Поиск изображений
