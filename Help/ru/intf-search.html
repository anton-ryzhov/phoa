{[%hh_hdr_ru:Окно поиска изображений]}

Программа PhoA обладает достаточно мощным средством выборки изображений. Для отображения окна поиска необходимо выбрать пункт меню Сервис | Поиск... или нажать клавишу F3 в режиме обзора.

Список критериев поиска

Вы вводите условия поиска в таблице критериев, которые определяются следующими данными:

  • Свойство изображения — один из поисковых элементов данных изображения, для которого вводится критерий.
  • Условие критерия — одно из условий сравнения свойства, набор возможных условий определяется типом данных.
  • Значение критерия — значение, с которым сравнивается свойство изображения.

Чтобы передвигаться по ячейкам таблицы, используются клавиши со стрелками. Для раскрытия выпадающего списка существует комбинация клавиш Alt+Вниз.

Значения условий хранят историю вводимых Вами значений.

Найти

Запускает поиск изображений.

Выбранные в результате поиска изображения (если они есть) попадают в папку Результаты поиска, которая отображается в корневом каталоге дерева групп. Для папки результатов поиска доступны те же операции, что и для всех групп изображений, за исключением добавления и удаления из неё изображений.

Сброс

Эта кнопка очищает все поля ввода и приводит все параметры к первоначальному виду. {[%hh_h2:Типы данных изображений]}

Согласно таблице данных изображения, PhoA использует следующие типы данных в свойствах изображений:
Тип данных Описание и критерии
Целое или натуральное число Данные представляют собой обычное целое число, которое можно проверить на равенство, неравенство и т. д.
Строка-имя файла Данные могут быть проверены на соответствие маскам. Маска составляется по стандартным правилам PhoA.
Строка Любые символьные данные. Поиск по строке или подстроке в любых вариантах. Поиск регистронезависим.
Пиксельный формат Не является поисковым типом.
Дата Данные проверяются по критериям, аналогичным тем, что используются у чисел.
Время Данные проверяются по критериям, аналогичным тем, что используются у чисел.
Список строк Это список ключевых слов. Слова можно вводить вручную или нажать кнопку "..." в поле ввода для выбора ключевых слов из списка. Критерии включают условия присутствия и отсутствия заданных (или всех) слов.

Программа выбирает те изображения, которые соответствуют всем критериям поиска одновременно (то есть здесь действует условие И).
