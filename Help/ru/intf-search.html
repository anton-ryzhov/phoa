{[%hh_hdr_ru:Окно поиска изображений]}

PhoA обладает достаточно мощным средством выборки изображений. Для отображения окна поиска необходимо выбрать пункт меню Сервис | Поиск... или нажать клавишу F3 в режиме обзора. {[%hh_h2:Элементы интерфейса]}
Искать Определяет область поиска изображений:
  • В фотоальбоме — поиск осуществляется среди всех имеющихся изображений.
  • В текущей группе — производить поиск только среди изображений текущей выбранной группы (недоступно, если нет выбранной группы).
  • В результатах поиска — искать среди результатов предыдущего поиска (доступно, только если имеется папка Результаты поиска).
Критерии поиска PhoA поддерживает два варианта поиска:
  • Простой поиск позволяет задать набор свойств изображений и наложить условия на их значения.
  • Поиск по выражению даёт возможность написания произвольного выражения фильтра, определяющего критерии поиска.
Найти Запускает поиск изображений. Выбранные в результате поиска изображения (если они есть) попадают в папку Результаты поиска, которая отображается в корневом каталоге дерева групп. Для папки результатов поиска доступны те же операции, что и для всех групп изображений, за исключением добавления и удаления из неё изображений.
