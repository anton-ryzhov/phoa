{[%hh_hdr_de:Optionen :: Anzeigemodus]}

Anzeigefenster immer im Vordergrund

Verhindert, dass das Fenster, auch wenn es inaktiv ist, von einem anderen uberlappt wird. Dies betrifft nur den Fenstermodus.

Vollbildmodus

Mit dem Befehl Anzeigen offnet das Programm das Bild im Vollbildmodus und verdeckt dabei die anderen Bildschirmelemente (auch die Taskleiste). Mit der Taste F oder der mittleren Maustaste kann diese Voreinstellung aufgehoben werden.

Mauszeiger im Vollbildmodus verbergen

Im Vollbildmodus wird der Mauszeiger ausgeblendet.

Hintergrundfarbe des Fensters

Bestimmt die Hintergrundfarbe des Anzeigefensters. (Voreinstellung schwarz).

Symbolleiste anzeigen

Aktiviert bzw. deaktiviert die Symbolleiste des Anzeigemodus. {[%hh_h2:Bildwechsel]}

Fenster an Bildgro?e anpassen

Das Fenster wird an die Bildgro?e angepasst (unter Berucksichtigung der aktuellen Skalierung). Betrifft nur den Fenstermodus.

Fenster auf dem Desktop zentrieren

Das Anzeigefenster wird auf dem Desktop zentriert. Betrifft nur den Fenstermodus.

Bild auf Fenstergro?e verkleinern

Ist ein Bild gro?er als das Anzeigefenster, wird es proportionell verkleinert (an das Fenster angepasst).

Bild auf Fenstergro?e zoomen

Ist ein Bild kleiner als das Anzeigefenster, wird es proportionell vergro?ert (an das Fenster angepasst).

Zyklische Ansicht

Ist diese Einstellung aktiviert, beginnt die Anzeige der Bilder wieder von vorne und wird endlos wiederholt.

Nachstes Bild im Voraus einlesen

Diese Funktion ermoglicht es, Bilder im Voraus einzulesen um somit schneller zum nachsten Bild wechseln zu konnen (im Hintergrund wird bereits das nachste Bild eingelesen). Dabei ist eine gro?ere Speicherkapazitat fur das Zwischenspeichern der im Hintergrund eingelesenen Bilddaten erforderlich.

Zuletzt angezeigtes Bild zwischenspeichern

Ist diese Einstellung aktiviert, wird das zuletzt angezeigte Bild nicht aus dem Speicher geloscht und somit kann schneller auf das vorherige Bild zuruckgegriffen werden. Dieser Modus erfordert eine gro?ere Speicherkapazitat fur das Zwischenspeichern der Bilddaten.

Skalieren um

Gibt die nachst gro?ere oder kleinere Skalierung an (mit den Tasten + oder - des numerischen Blocks oder mit der Scroll-Funktion der Maus bei gedruckter Strg Taste).

In der Titelleiste anzeigen

Ermoglicht die Auswahl der in der Titelleiste angezeigten Bildeigenschaften.

Resampling Filter

Wahrend dem Anzeigen der Bilder werden die Interpolationsfilter bestimmt, die fur das Resampling (Skalieren) der Quellbilder verwendet werden. {[%hh_h2:Info anzeigen]}

Bildeigenschaften

Bestimmt, welche Bildeigenschaften im Anzeigemodus angezeigt werden.

Schrift

Bestimmt die Schrift der Bildinformation im Anzeigemodus.

Hintergrundfarbe

Mit dieser Funktion wird die Hintergrundfarbe der Informationsleiste eingestellt. Siehe nachste Option.

Hintergrund Deckfahigkeit (0-255)

Der Wert der Deckfahigkeit reicht von 0 (transparent, d.h. es gibt keinen Hintergrund) bis 255 (schwarz); damit kann die Durchsichtigkeit der Informationsleiste eingestellt werden. {[%hh_h2:Diashow]}

Anzeigezeit, ms

Bestimmt die Zeitintervalle (in Millisekunden) zwischen den Bildern einer Diashow. Der Diashow-Modus wird durch das Betatigen der Pause-Taste im Anzeigemodus oder dem Diashow-Button in der Symbolleiste bzw. mittels Kontextmenu aktiviert.

Zyklisch

Ist diese Einstellung aktiviert, greift PhoA automatisch - nach dem Anzeigen des letzten Bildes im Diashow-Modus - auf das erste Bild in der Gruppe zuruck. Ist die Einstellung nicht aktiviert, wird die Diashow nach dem letzten Bild beendet. {[%hh_seealso_de]} Optionen :: Allgemein
