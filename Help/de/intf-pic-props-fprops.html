Diese Seite zeigt eine Liste der Dateien ausgewahlter Bilder an. Wenn Sie den Dateinamen aufklappen bzw. erweitern, konnen Sie die Dateiattribute wie (Dateityp, Gro?e, Erstellungs-, Bearbeitungs- und Zugriffszeit usw.) sehen.

Beim Anklicken mit der rechten Maustaste konnen Sie das Standard Explorer Kontextmenu fur diese Datei offnen.

NB: Im Dialog Optionen kann eingestellt werden, dass alle Eigenschaften automatisch aufgeklappt dargestellt werden.
