{[%hh_hdr_de:Browser Modus Menubefehle]} {[%hh_h2:Menu Datei]}

Dieses Menu enthalt Befehle fur die Bearbeitung der Fotoalbumdatei:

  • Es wird automatisch ein neues Fotoalbum erstellt, wenn das Programm gestartet wird oder wenn man im Menu auf Datei | Neu klickt. Es loscht alle Gruppen und Bilder wahrend die Eigenschaften des Fotoalbums zuruckgesetzt werden.
  • Um eine bereits angelegte Fotoalbumdatei zu offnen, klickt man im Menu auf Datei | Offnen.... Dieser Befehl offnet ein herkommliches Dialogfeld, wo man eine Datei auswahlen kann (mit dem Dateiformat .phoa).
  • Um das Fotoalbum in der derzeit geoffneten Datei zu speichern, klickt man im Menu auf Datei | Speichern.
  • Um es in einer anderen Datei zu speichern, klickt man im Menu auf Datei | Speichern unter....
  • Mit dem Befehl Datei | Beenden konnen eventuelle Anderungen gespeichert werden und anschlie?end wird das Programm beendet.
  • Darunter findet man zuletzt geoffnete Dateien (die Anzahl dieser gespeicherten Eintrage kann man unter Optionen) einstellen.
{[%hh_h2:Menu Bearbeiten]}

Dieses Menu beinhaltet Befehle zur Bearbeitung des Fotoalbums:

  • Der Befehl Bearbeiten | Ruckgangig Anderung im Menu macht die zuletzt vorgenommene Anderung im Fotoalbum ruckgangig.
  • Der Befehl Berabeiten | Ruckgangig Verlauf im Menu enthalt die Liste der vorgenommenen Anderungen, wobei man mehrere aufeinanderfolgende Anderungen gleichzeitig ruckgangig machen kann.
  • Der Befehl Bearbeiten | Ausschneiden im Menu schneidet markierte Bilder aus und speichert sie in der Zwischenablage (man kann diese Bilder spater zu anderen Bildergruppen hinzufugen).
  • Auch der Befehl Bearbeiten | Kopieren im Menu ermoglicht es markierte Bilder in die Zwischenablage zu kopieren.
  • Der Befehl Bearbeiten | Einfugen fugt Bilder aus der Zwischenablage (falls vorhanden) in die derzeit geoffnete Gruppe ein. Naturlich konnen einer Gruppe nur neue Bilder hinzugefugt werden; alle doppelten Bilder werden ignoriert. Wird es in den Optionen so definiert, erfahrt man von PhoA wieviele Bilder man tatsachlich eingefugt hat.
  • Das Fotoalbum ist die oberste Stufe in der Einteilung der Bildergruppen, und steht daher als erstes in der Baumansicht. Das Fotoalbum kann Textbeschreibungen beinhalten und definiert auch Thumbnail Parameter. Man kann diese im Dialog Eigenschaften Fotoalbum andern, das durch Klicken auf Bearbeiten | Bearbeiten... im Menu erscheint, wobei Fotoalbum in der Baumansicht markiert sein muss.
  • Eine Bildergruppe kann umbenannt werden indem man im Menu auf Bearbeiten | Bearbeiten... klickt, jedoch muss die jeweilige Gruppe in der Baumansicht markiert sein; man kann aber auch einmal auf eine markierte Gruppe klicken oder die F2 Taste drucken. Um den Bearbeitungsvorgang abzubrechen, druckt man die Esc Taste und um den neuen Gruppennamen zu bestatigen, druckt man die Enter Taste.
  • Um eine neue Gruppe hinzuzufugen klickt man im Menu auf Bearbeiten | Gruppe hinzufugen. Diese Gruppe ist eine Untergruppe der derzeit markierten Gruppe. Wird Fotoalbum in der Baumansicht markiert, ist die Untergruppe in der Baumansicht zu sehen. Der Name dieser Gruppe ist anfanglich "Neue Gruppe", den man dann aber bearbeiten und durch das Drucken der Enter Taste bestatigen kann.
  • Eine Gruppe kann geloscht werden, indem man im Menu auf Bearbeiten | Loschen klickt, wenn diese in der Baumansicht markiert ist.
  • Um Bilder zu einer Gruppe hinzuzufugen, klickt man im Menu auf Bearbeiten | Bild hinzufugen.... Dadurch wird das Fenster Bilder hinzufugen geoffnet.
  • Um Bilder zu ordnen (sortieren), klickt man im Menu auf Bearbeiten | Bilder sortieren.... Dadurch wird das Fenster Bilder sortieren geoffnet.
  • Der Befehl Bearbeiten | Alles auswahlen markiert alle Bilder der derzeit geoffneten Gruppe.
  • Der Befehl Bearbeiten | Nichts auswahlen entfernt die Markierung aller Bilder.
  • Um zu dem Anzeigemodus zu wechseln, klickt man entweder im Menu auf Bearbeiten | Anzeigen oder klickt zweimal auf ein Bild.
{[%hh_h2:Menu Anzeigen]}

In diesem Menu kann man die Anzeigeoptionen des Programms einstellen:

  • Die Symbolleiste kann man im Menu Anzeigen | Symbolleiste aktivieren und deaktivieren.
  • Die Statusleiste kann man im Menu Anzeigen | Statusleiste aktivieren und deaktivieren. Das Programm speichert diese Einstellungen automatisch und ruft sie beim nachsten Offnen wieder auf.
  • Darauf folgen im Menu die Einstellungen fur die Fotoalbumansichten:
    • Klickt man auf Bildgruppen wird die ursprungliche Baumansicht der Bildgruppen wieder im linken Programmfenster hergestellt.
    • Klickt man im Menu auf Neue Ansicht... offnet sich der Dialog Eigenschaften Fotoalbum Ansichten um einen neuen Anzeigemodus einzustellen.
    • Klickt man im Menu auf Ansicht loschen wird die momentan aktive Ansicht geloscht (falls die Ansicht und nicht die Bildgruppen aktiviert sind).
    • Klickt man im Menu auf Ansicht bearbeiten... offnet sich der Dialog Eigenschaften um die momentan aktive Ansicht zu bearbeiten (falls die Ansicht und nicht die Bildgruppen aktiviert sind).
    • Klickt man im Menu auf Gruppe erstellen... offnet sich der Dialog Bildgruppe auswahlen, wodurch man die genaue Struktur der aktiven Ansicht einer ausgewahlten Bildgruppe des Fotoalbums hinzufugen kann.
{[%hh_h2:Menu Tools]}

Hier findet man optionale Extra-Befehle:

  • Klickt man im Menu auf Tools | Optionen... offnet sich das Dialogfeld PhoA Optionen Browser Modus.
  • Klickt man im Menu auf Tools | Statistiken... werden die statistischen Daten des Fotoalbums angezeigt.
  • Klickt man im Menu auf Tools | Thumbnails erneuern... offnet sich der Dialog Thumbnails erneuern. Dort kann man fur die ausgewahlten Bilder, fur alle Bilder in der aktuellen Gruppe oder alle Bilder im Fotoalbum die Thumbnails erneuern. Dieser Vorgang ist jedesmal notig, wenn man die Thumbnails Einstellungen im Dialogfeld Einstellungen Fotoalbum andert.
  • Klickt man im Menu auf Tools | Bildoperationen... kann man im Dialog Bildoperationen Bilder ausschneiden und in eine andere Gruppe verschieben, kopieren oder einfugen.
  • Eine der wichtigsten Eigenschaften von PhoA ist die Suche nach Bildern in dem man auf Tools | Suchen... klickt. Fur weitere Informationen hier klicken.
{[%hh_h2:Menu Hilfe]}

Dieses Menu enthalt drei Untermenus:

  • Klickt man im Menu auf Hilfe | Uber... erhalt man Informationen uber die Version, Urheberrechte, Updates und Kontaktmoglichkeiten.
  • Klickt man im Menu Hilfe | Hilfethemen offnet sich das PhoA Hilfesystem.
  • Klickt man im Menu auf Hilfe | PhoA Website offnet sich der Internetbrowser und die Homepage des Programms wird gestartet.
