{[%hh_hdr_de:Dialog Bearbeiten Eigenschaften Bild :: Anzeigen]}

Hier kann man jedes ausgewahlte Bild in der gewahlten Skalierung sehen. Das Bild (wenn mehr als ein Bild ausgewahlt ist) kann in der Dropdown Liste ausgewahlt werden.

Zoom in

Vergro?ert das Bild je nach Einstellungen in Optionen Anzeigemodus.

Zoom out

Verkleinert das Bild je nach Einstellungen in Optionen Anzeigemodus.

Tatsachlicher Zoom

Damit kann die Skalierung auf 1:1 eingestellt werden.

Dem Fenster anpassen

Hier wird das Bild so skaliert, dass es in das Fenster passt. Dies ist die Standardeinstellung.

NB: Dieser Befehl kann auch uber das Popup-Fenster ausgefuhrt werden, das beim Drucken der rechten Maustaste erscheint.

Man kann Bilder scrollen (falls das Bild gro?er als das Fenster ist), indem die linke Maustaste gedruckt gehalten wird. Zudem konnen Sie die Bilder mit dem Mausrad zoomen.

NB: Um das Anzeigen der Bilder zu beschleunigen, wird stets die Nearest Neighbor-Methode zum Resampling verwendet.
