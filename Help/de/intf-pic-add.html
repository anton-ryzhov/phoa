{[%hh_hdr_de:Bild hinzufügen Assistent]}

Mit dieser Funktion können Sie Bilder zu einer gerade ausgewählten Gruppe hinzufügen (auch zur Hauptgruppe, d.h. zum Fotoalbum selbst). Wenn keine Gruppe ausgewählt wurde, kann diese Funktion nicht angewendet werden. {[%hh_h2:Dialog 1: Dateien und Ordner auswählen]}

Wählen Sie zuerst Bilddateien und/oder Ordner, die Bilddateien enthalten, aus.

Liste der Dateien/Ordner

Wählen Sie jene Elemente, die Sie hinzufügen wollen, mithilfe der Tastatur oder der Maus plus Shift-Taste oder Strg-Taste aus.

Inklusive Unterordner

Wenn dieses Kontrollkästchen aktiviert ist und Ordner im Baum ausgewählt sind, werden nicht nur Dateien aus diesen Ordnern hinzugefügt, sondern auch Dateien aus sämtlichen Unterordnern.

Erweitert

Zeigt den Dateiauswahlfilter an bzw. verbirgt ihn. Mit diesem können Sie die Dateiauswahl durch bestimmte Kriterien eingrenzen (siehe unten).

Vorhandensein im Fotoalbum

Sie haben die Möglichkeit, Bilddateien nach ihrem Vorhandensein im Fotoalbum zu filtern:

  • Nicht berücksichtigen - der Filter wird unterdrückt.
  • Nur Dateien, die noch nicht im Fotoalbum vorhanden sind - es werden nur Dateien hinzugefügt, die noch nicht in der Bilderliste des Fotoalbums sind.
  • Nur Dateien, die schon im Fotoalbum vorhanden sind - es werden nur Dateien hinzugefügt, die bereits im Fotoalbum sind.

Dateimaske

Dieser Filter fügt nur Dateien mit einem bestimmten Dateinamen und/oder Format hinzu. Die Maske ist wie üblich aufgebaut: ? ersetzt jedes Zeichen und * ersetzt eine beliebige Anzahl von Zeichen; außerdem können Sie in eckigen Klammern Zeichenketten spezifizieren, wie Sie es auch im Dialog Bilder suchen tun können.

Sie können mehr als eine Maske spezifizieren, indem Sie sie durch einen Strichpunkt trennen. In diesem Fall wird eine Datei zugelassen, wenn sie den Kriterien einer der Masken entspricht.

Datei bearbeitet zwischen

Sie können Zeitspannen (Datum und/oder Uhrzeit) festlegen, in denen die Datei bearbeitet wurde. So können Sie z.B. Dateien auswählen, die vor oder nach einem bestimmten Datum gespeichert worden sind. Lassen Sie das Feld leer, ist der Filter nicht aktiv. {[%hh_h2:Dialog 2: Dateien überprüfen]}

In diesem Fenster können Sie durch "Markieren" oder "Markierung aufheben" entscheiden, welche der im vorhergehenden Dialog ausgewählten Dateien hinzugefügt werden sollen. Wenn Sie selbst nichts ändern, sind von vornherein alle Dateien markiert.

Sie können die Markierung mit der Maus oder mit der Leertaste vornehmen. Letztere hebt die Markierung aller blau markierten Dateien auf bzw. aktiviert sie. Das Popup-Menü (aufgerufen durch Klicken der rechten Maustaste) bietet zusätzliche Optionen.

NB: Dieses Dialog wird nur angezeigt, wenn die Standardoptionen verwendet werden. Sie können das Anzeigen dieses Fensters unter Optionen - Dialoge verhindern, dann wird es übersprungen und nach dem Dialog Dateien/Ordner auswählen startet sofort der Prozess Bilder hinzufügen.
{[%hh_h2:Dialog 3: Bilder hinzufügen]}

Dieses Fenster wird angezeigt, während die ausgewählten Bilddateien hinzugefügt werden. Das Programm zeigt den Fortschritt des Prozesses und die Zahl der aufgetretenen Fehler an. Sie können sich diese Fehler dann im Protokoll ansehen, sofern einige Bilder erfolgreich hinzugefügt wurden.

Außerdem wird von dem zuletzt hinzugefügten Bild ein Thumbnail angezeigt.

NB:
  • Der Prozess Bilder hinzufügen wird parallel umgesetzt; Sie können diesen Prozess nur zwischen der Verarbeitung zweier Bilder unterbrechen, deshalb reagiert das Programm verzögert auf den Befehl Unterbrechen.
  • Außerdem können Sie dieses Fenster nicht schließen, während Bilder hinzugefügt werden. Sie müssen auf Unterbrechen klicken, bevor das Fenster geschlossen werden kann.
  • Beim Hinzufügen von Bildern wird die Funktion Automatisches Auslesen und Einfügen der Bildeigenschaften aktiv. Sie können in Optionen - Dialoge festlegen, welche Bilddaten übernommen werden sollen; im Protokoll wird angezeigt, welche Daten übernommen wurden.

Unterbrechen

Wenn Sie diese Schaltfläche anklicken, wird der Prozess Bilder hinzufügen unterbrochen. Diese Schaltfläche wird dann zu Weiter, und durch Anklicken nehmen Sie den Prozess wieder auf.

Wenn der Prozess abgeschlossen ist, schließt das Programm das Fenster des Assistenten (nur wenn keine Fehler aufgetreten sind und in den Optionen bestimmt wurde, das Protokoll in einem solchen Fall nicht anzuzeigen) oder es geht zum Protokoll über. {[%hh_h2:Dialog 4: Protokoll von Bilder hinzufügen]}

Das Protokoll zeigt alle an den Bilddateien durchgeführten Operationen der Reihe nach an. An der Kennzeichnung der (ausgewählten) Dateien ist erkennbar, ob der Prozess erfolgreich war oder gescheitert ist. Wenn letzteres der Fall ist, erscheint eine Fehlerbeschreibung neben dem Dateinamen, im Erfolgsfall werden die automatisch übernommenen Bilddaten angezeigt.

Nur Fehler anzeigen

Wenn dieses Kontrollkästchen aktiviert ist, zeigt das Protokoll nur die fehlgeschlagenen Versuche an, sonst werden auch die erfolgreichen angezeigt. Diese Option ist nützlich, wenn Sie einen Fehler in einer großen Anzahl hinzugefügter Dateien finden müssen.

In Datei speichern...

Ermöglicht Ihnen, das Protokoll als Textdatei zu speichern.

NB:
  • Wenn das Protokoll gespeichert wird, werden alle Versuche festgehalten, ungeachtet des Kontrollkästchens Nur Fehler anzeigen.
  • Fehlgeschlagene Versuche werden mit Symbolen gekennzeichnet [!], erfolgreiche vorne mit [+]. So können Sie Fehler im Protokoll leicht finden.

Sie können das Protokoll anschauen, bevor alle Dateien hinzugefügt worden sind, indem Sie den Hinzufügungsprozess unterbrechen und dann die Schaltfläche Weiter anklicken. Danach können Sie mit Zurück den Prozess wieder aufnehmen.
