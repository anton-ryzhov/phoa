{[%hh_hdr_de:Dialog Bearbeiten Eigenschaften Bild :: Daten]} {[%hh_h2:Bildeigenschaften]}

Dieser Dialog erlaubt das Bearbeiten der meisten bearbeitbaren Bilddaten:

  • Datum - Bilddatum. Das Datumsformat hangt von den regionalen Windows-Einstellungen ab. Sind keine Daten zum Bilddatum verfugbar, bleibt dieses Feld frei.
  • Zeit - Bildzeit. Sind keine Daten zur Bildzeit verfugbar, bleibt dieses Feld frei.
  • Ort - Ort der Aufnahme. Eine Auswahlliste zeigt alle bereits gespeicherten Informationen an.
  • Filmnummer/-name - Nummer, die der Film wahrend der Ausarbeitung erhalt, oder der Name, den Sie selbst dem Film gegeben haben. Eine Auswahlliste zeigt alle bereits gespeicherten Informationen an.
  • Frame Nummer - keine Angaben.
  • Autor - Name des Autors oder des Fotografen. Eine Auswahlliste zeigt alle bereits gespeicherten Informationen an.
  • Beschreibung - eine kurze Bildbeschreibung, die wahrend des Betrachtens der Bilder angezeigt werden kann.
  • Anmerkungen - alle Daten, die in kein anderes Feld passen.
  • Medienname/-code - Medienname der Bilddatei (z.B.Nummer der CD-ROM ). Eine Auswahlliste zeigt alle bereits gespeicherten Informationen an.

Detailliertere Informationen zu Bilddaten finden Sie hier. {[%hh_h2:Eine Vielzahl von Bildern gleichzeitig bearbeiten]}

PhoA ermoglicht das gleichzeitige Bearbeiten von Bilddaten fur eine beliebige Zahl von Bildern. Dafur gilt die folgende einfache Vorgangsweise:

  • Sind einige Bildeigenschaften bei allen ausgewahlten Bildern gleich, bleibt das dazugehorige Eingabefeld wei? (bzw. hat die Farbe des Bildschirmhintergrunds).
  • Ist eine Bildeigenschaft bei den ausgewahlten Bildern verschieden, bleibt das dazugehorige Eingabefeld grau (oder hat die Farbe des 3D-Objekts). Verandern Sie die Daten in diesem Feld, so werden diese Daten fur alle ausgewahlten Bilder ubernommen und das Feld wird wei?. Sonst bleiben die Bilddaten unverandert (und naturlich auch verschieden).

Anschlie?end kann eine Bildersuche aufgrund dieser Daten vorgenommen werden.
