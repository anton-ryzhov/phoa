{[%hh_hdr_de:Optionen :: Dialoge]} {[%hh_h2:Bestatigungen]}

Vor dem Loschen von Gruppen

Wenn aktiviert, verlangt das Programm eine Bestatigung, bevor eine Gruppe geloscht wird.

Vor dem Loschen von Bildern

Wenn aktiviert, verlangt das Programm eine Bestatigung, bevor die ausgewahlten Bilder geloscht werden.

Vor dem Loschen von Ansichten

Wenn aktiviert, verlangt das Programm eine Bestatigung, bevor eine Ansicht geloscht wird.

Vor dem Speichern von Dateien alterer Versionen

Wenn aktiviert, warnt das Programm vor moglichem Datenverlust, bevor die Datei einer alteren als der aktuellen Version gespeichert wird.

Vor dem Beenden (wenn das Fotoalbum nicht verandert wurde)

Wenn aktiviert, verlangt das Programm vor dem Beenden eine Bestatigung, auch wenn nichts an dem Fotoalbum verandert wurde (wenn etwas am Fotoalbum verandert wurde, verlangt das Programm immer eine Bestatigung). {[%hh_h2:Benachrichtigungen]}

Nach dem Kopieren von Bildern mittels Drag'n'Drop

Wenn aktiviert, benachrichtigt das Programm nach dem Kopieren von Bildern uber die tatsachlich kopierte Anzahl von Bildern.

Nach dem Verschieben von Bildern mittels Drag'n'Drop

Wenn aktiviert, benachrichtigt das Programm nach dem Verschieben von Bildern uber die Anzahl der tatsachlich verschobenen Bilder.

Nach dem Einfugen von Bildern aus der Zwischenablage

Wenn aktiviert, benachrichtigt das Programm nach dem Einfugen der Bilder uber die tatsachlich eingefugte Anzahl von Bildern. {[%hh_h2:Bilder hinzufugen Assistent]}

'Datei auswahlen' Seite immer uberspringen

Wenn aktiviert, beginnt der Bilder hinzufugen Assistent PhoA sofort mit dem hinzufugen von Bildern.

Logbuch nur anzeigen wenn Fehler aufgetreten sind

Wenn aktiviert, wird das Logbuch nur angezeigt, wenn Fehler aufgetreten sind. Ansonsten schlie?t sich das Assistentenfenster sofort nach dem Hinzufugen. Wenn deaktiviert, wird das Logbuch nach jedem Hinzufugen angezeigt.

Bilddatum automatisch ubernehmen von:

Das Bilddatum wird fur neu hinzugefugte Bilder automatisch ubernommen.

Datum, falls bereits angegeben, uberschreiben

Das Programm ubernimmt automatisch das Bilddatum (uberschreibt das Datum) auch wenn das Bilddatum bereits eingegeben wurde.

Bildzeit automatisch ubernehmen von:

Das Bildzeit wird fur neu hinzugefugte Bilder automatisch ubernommen.

Zeit, falls bereits angegeben, uberschreiben

Das Programm ubernimmt automatisch die Bildzeit (uberschreibt die Zeit) auch wenn die Bildzeit bereits eingegeben wurde. {[%hh_h2:Bildeigenschaften Dialog]}

Dateieigenschaften beim Anzeigen aufklappen

Wenn aktiviert, werden die Dateieigenschaften auf der Eigenschaftenseite beim Anzeigen komplett aufgeklappt.

Metadaten beim Anzeigen aufklappen

Wenn aktiviert, werden die Metadaten auf der Metadatenseite beim Anzeigen komplett aufgeklappt. {[%hh_seealso_de]} Optionen :: Allgemein
