{[%hh_hdr_de:Dialog Bearbeiten Eigenschaften Bild :: Metadaten]}

Diese Seite zeigt die in der Bilddatei enthaltenen Metadaten. Die Metadaten werden in einer Baumansicht dargestellt, durch Anklicken wird der Baum aufgeklappt.

Wenn die Datei keine Metadaten enthalt, erscheint rechts neben dem Dateinamen die entsprechende Nachricht.

Auf der unteren Leiste gibt die Beschreibung zusatzliche Information fur die ausgewahlte Zeile. Die Leiste ist in der Grundeinstellung sichtbar, kann aber geschlossen werden, indem man auf das X rechts oben in der Leiste klickt. Um die Leiste wieder aufzurufen, klickt man mit der rechten Maustaste auf die gewunschte Zeile im Kontextmenu.

NB:
  • Im Dialogfeld Optionen Dialoge kann man die Einstellung "Metadaten beim Anzeigen aufklappen" aktivieren.
  • Wenn man ein neues Bild zu dem Fotoalbum hinzufugt, werden in die Metadaten fur das Automatisches Ubernehmen der Bildeigenschaften genutzt. Dazu muss die entsprechende Einstellung in Optionen Dialoge aktiviert werden.
