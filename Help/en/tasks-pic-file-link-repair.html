{[%hh_hdr_en:Common tasks :: Picture file link repairing]}

As is well known, PhoA photo album groups contain links to pictures, while each picture refers a corresponding image file.

A saved photo album file holds relative links to picture files whenever possible. This means, if you move a photo album file together with image files (keeping their relative placement unchanged), all picture file links will remain valid.

To move image files independently from photo album file the program provides the special move picture files operation.

But when you move an image file outside the program, the corresponding link gets broken, or lost, i. e. points to nowhere.

You can fix this in the two ways:

  • A "soft" method: you indicate a new placement of the file.
  • A "hard" method: you delete all the images linked to non-existent files. In that case you lose all the information entered for those pictures.
{[%hh_h2:Changing picture file link]}
  1. Select thumbnails of the images you would like to relink.
  2. Select the {[%hh_ii:Edit]} Edit | Edit... menu item: this will show the Picture properties dialog.
  3. Open the File properties page of the dialog.
  4. Select a file in the list and click the {[%hh_ii:Edit]} Change file... toolbar button.
  5. In the appeared dialog select a new location of the picture file and click Open.
  6. Repeat items 4 and 5 for each of the broken links.
  7. Save changes to file links by clicking OK in the Picture properties dialog.
{[%hh_h2:Delete picture files operation]}

To remove all images having no valid file linked perform the following steps.

  1. Launch the File operations wizard. You do it by selecting the {[%hh_ii:FileOps]} Tools | File operations... menu item.
  2. On the first page of the wizard select Delete pictures and files and click Next.
  3. Select All pictures in the photo album, and in the File link validity filter select Only pictures having NO valid (existent) file linked.

    WARNING! A little mistake here can cause that all of your files will be deleted so be careful!

  4. As long as we are deleting only pictures with no existent file linked, option for removing to the Recycle Bin makes no sense. After clicking Next the program will start removing pictures.
