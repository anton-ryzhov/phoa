{[%hh_hdr_en:System requirements]}

For the program to install you need an operating system, some disk space and a bit of RAM. To be more specific, PhoA runs under Windows 95/98/ME/NT/2000/XP, takes approximately 2 MB of disk space and requires at least 16 MB of installed RAM. Because entire photo album file being opened is loaded into RAM, when opening file containing a lot of pictures (1000+) it is recommended to have adequate amount of RAM (32-64 MB on Windows 95/98/ME/NT, 128 MB on Windows 2000 & XP).
