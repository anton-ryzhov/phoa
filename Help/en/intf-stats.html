{[%hh_hdr_en:Photo album statistics]}

This function calculates and displays summary information on the photo album, its groups and its pictures, and displays it.

Basic photo album statistics items are:
Name Description
Photo album file Photo album file properties are presented similar to picture file properties. Moreover, here you can find Photo album file revision item, displaying version of file loaded.
Number of distinct pictures Number of distinct pictures in the photo album.
Number of nested groups Number of groups in the photo album.
Number of pictures in th group Number of pictures explicitly owned by the photo album (the root node of group tree).
Total number of pictures Total number of links to the pictures (not needed to be distinct) in the photo album group and all nested groups.
Total number of distinct pictures Total number of links to distinct pictures in the photo album group and all nested groups.
Total picture files size Sum of file sizes of pictures in the photo album group and all nested groups.
Average picture file size Average picture file size from pictures in the photo album group and all nested groups.
Total thumbnails size Sum of data sizes of picture thumbnails from pictures in the photo album group and all nested groups.
Average thumbnail size Average thumbnail data size from pictures in the photo album group and all nested groups.
Largest picture file Name of the picture file having the grestest size from pictures in the photo album group and all nested groups.
Size of the largest file Size of the file from the previous item.
Smallest picture file Name of the picture file having the least size from pictures in the photo album group and all nested groups.
Size of the smallest file Size of the file from the previous item.

If there is a picture group selected in the tree then there follows all items listed above (except file name and number of pictures in the photo album) relative to this group. {[%hh_seealso_en]} Browse mode
