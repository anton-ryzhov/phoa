{[%hh_hdr_en:Program settings :: View mode]}
View window always on top Does not allow the window to fall behind other windows when it's inactive. Affects only the window mode.
Fullscreen mode When entering view mode the program will set up the fullscreen mode when all other screen elements are hidden (including the Taskbar). This can be overriden with the F key or by clicking middle mouse button.
Keep mouse cursor over the toolbar button on window resize If on, the mouse cursor stays at the same point of the viewer toolbar when the window resizes. This allows you to click buttons (such as Next picture or Zoom out) multiple times without having to move the mouse.
Hide mouse cursor in fullscreen When entering fullscreen mode the program will turn off mouse pointer.
Window background color Determines color for filling the viewing window background. The default is black.
Show toolbar Enables or disables view mode toolbar.
Picture advancing
Fit window to image Resizes window so that it fits the picture (considering current scale). Affects only the window mode.
Center window on the desktop Places viewing window in the center of the desktop. Affects only the window mode.
Shrink image to fit If a picture is larger than the viewing window, it will be proportionally reduced (to fit the window).
Zoom image to fit If a picture is smaller than the viewing window, it will be proportionally enlarged (to fit the window).
Cyclic viewing If on, the viewing sequence wraps around.
Predecode next picture in advance This option enables background picture decoding thus allowing quick advancing to the next picture (next picture is being decoded while viewing current one). This mode requires more memory for storing predecoded image data.
Cache behind viewed pictures If this option is enabled, last viewed picture is not removed from the memory thus allowing you to quckly revert to previous picture. This mode requires more memory for storing cached image data.
Scale by Determines the one step of scaling change (with + and - keys on numeric keypad or with mouse wheel when Ctrl depressed).
Display in window caption Allows you to choose which picture properties are to be displayed in the window caption bar.
Picture resampling filter Determines interpolation filter used for resampling (scaling) source images when viewing them.
View info
Show picture info Enables or disables displaying the information from the properties described below. Can be overriden in the view mode by pressing the F2 key.
Picture properties Specifies which picture properties will be shown in the view mode.
Font Determines which font will be used for drawing the picture info.
Background color Specifies the color of information bar background. See the next option.
Background opacity Value ranging from 0 (transparent) to 255 (opaque). Allows to adjust the translucence of information bar background.
Slide show
Display time Determines time intervals (in milliseconds) between picture changes in Slide Show mode (this mode is activated by pressing the Pause key in the view mode, with the toolbar button or by context menu).
Cyclic When on, PhoA automatically rewinds to the first picture in the group after showing the last in Slide Show mode, else Slide Show is deactivated once the last picture is reached.
