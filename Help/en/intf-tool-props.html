{[%hh_hdr_en:Tool properties dialog]}

This window allows you to view or edit a tool properties. To display it, select Edit... menu item on Tools page of Program settings dialog.

File masks

Determine files the tool is applicable to.

Kind

Specifies the tool's kind.

Name

Specifies a menu item caption for the tool. You can mark a character a hotkey by prepending it with an ampersand (&). To include the ampersand itself you'll need to duplicate it.

Hint

The hint is displayed in the status bar. If the string contains pipe character (|) the status bar displays only the text following it while the preceding text is used for toolbar button's tooltip.

Command (application)

Is applicable only for custom action. Specifies an executable file to pass picture file names to. You may use Browse... button to select a file.

Parameters

Specify additional command line parameters being passed to the application.

Window

Determines main application's window state when it starts.

Folder

Specifies working folder of the application. Browse... button simplifies folder selection.

Tool usage

Determines which menus will contain the tool items. {[%hh_seealso_en]} Tools
