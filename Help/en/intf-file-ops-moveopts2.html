{[%hh_hdr_en:Picture file operations wizard :: Copy/move options page 2]}

This wizard page configures parameters for the following file operations:

  • Copying picture files into a specified folder
  • Moving picture files into a specified folder
{[%hh_h2:Controls]}
If the original file does not exist: Is applicable for Move files operation only and determines the program behaviour when it finds that the source file doesn't exist:
  • Treat as an error - logs the error and skips the picture.
  • Update the link only if the target file exists - if a file found as a target one exists the picture will be linked to it; otherwise the picture will be skipped. This mode is useful in cases you have moved files from outside the PhoA, and wish to restore the valid links. In that case you should specify the folder where new files reside (but take into account their former and new folder structure).
  • Update the link anyway - the link to picture file will be changed disregarding the source and target files existence.
Delete original files Is applicable for Move files operation only. When on, the successful file copying and link updating will be followed by deletion of the file.
Delete to the Recycle Bin Is applicable for Move files operation only.
  • If this option is off, all the files are being wiped off and cannot be restored.
  • If the option is on, the files are being moved into the Recycle Bin rather than deleted (if this is possible on the given machine).
If the target file already exists Specifies the overwrite mode for existing files:
  • Skip the current picture - if the target file exists the program will consider this an error, log this and leave the file intact.
  • Prompt to overwrite the file - the program will ask you to overwrite the existing file. If you decide not to overwrite the file the picture will also be logged as failed.
  • Always overwrite the file - the file will be overwritten with the new one without any notifications.
CD/DVD options
Set up CD/DVD creation options as well If on, the Next > button click will cause the wizard to advance to the CD/DVD creation options page, or otherwise the program will start processing pictures (after confirmation, if this is specified in the Program settings).
