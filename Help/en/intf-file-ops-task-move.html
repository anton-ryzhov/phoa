{[%hh_hdr_en:Picture file operations wizard :: Move picture files operation]}

This operation is basically the same as copy files operation, except for the two points:

  • During the move process picture file links are being changed to the new locations.
  • The user can opt to delete the original files (completely or to the Recycle Bin).

Use this operation to reorganize files in your photo album, eg. to move them into another folder, or onto another drive.

To perform this operation you need to start the Picture file operations wizard and on the Task selection page select Move picture files to a folder.

The next pages will be:

  • Picture selection
  • Destination folder selection
  • Copy/move options (page 1)
  • Copy/move options (page 2)
  • CD/DVD creation options
  • Operation process
  • Operation log
