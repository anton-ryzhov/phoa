{[%hh_hdr_en:Browse mode menu commands :: File menu]}

This menu contains commands for photo album file operations.
Menu Item Description
{[%hh_ii:New]} New New photo album creation. This erases all groups, pictures and views, and photo album properties are reset to their defaults.
{[%hh_ii:Open]} Open... Opens a standard File open dialog allowing you to select a photo album file (with the .phoa extension).
{[%hh_ii:Save]} Save Saves the photo album to the file currently open. If this is a new photo album, this item executes Save as... instead.
{[%hh_ii:SaveAs]} Save as... Displays a standard File save dialog to save the photo album into another file.
{[%hh_ii:SaveSettings]} Save current settings to file... Displays the Save current settings dialog allowing you to store the current program settings into an .ini file, details.
{[%hh_ii:LoadSettings]} Load settings from file... Displays the Load program settings dialog for restoring program settings from an .ini file, details.
{[%hh_ii:Exit]} Exit Confirms whether you wish to save changes if there were any (or if it is configured in the Confirmations settings), and then exits the program.
