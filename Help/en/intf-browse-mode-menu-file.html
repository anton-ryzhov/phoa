{[%hh_hdr_en:Browse mode menu commands :: File menu]}

This menu contains commands for photo album file operations:

  • {[%hh_ii:New]} New photo album is created automatically on the program startup or when you select the File | New menu item. It erases all groups, pictures and views while the photo album properties are being reset to the defaults.
  • {[%hh_ii:Open]} To open a photo album file you created before, you select the File | Open... menu item. This command brings up a standard dialog allowing you to select a file (with the .phoa extension).
  • {[%hh_ii:Save]} To save photo album in the file currently open you select the File | Save menu item.
  • {[%hh_ii:SaveAs]} To save it into another file - the File | Save as... menu item.
  • {[%hh_ii:SaveSettings]} The File | Save current settings to file... menu item displays a Save current settings dialog to an .ini file, details.
  • {[%hh_ii:LoadSettings]} The File | Load settings from file... menu item displays a Load program settings dialog from an .ini file, details.
  • {[%hh_ii:Exit]} The File | Exit menu item confirms whether you wish to save changes if there were any (or if it is selected in the Confirmations settings) and exits the program.
