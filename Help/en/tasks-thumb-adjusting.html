{[%hh_hdr_en:Common tasks :: Configuring thumbnail dimensions and quality]}

Thumbnails created by the program are by default of size 150x150 pixels and quality about 40% max.

These values are stored on a per-album basis, and you can modify them in photo album properties. {[%hh_h2:Changing thumbnail parameters]}

  1. Highlight the root node in the group tree (Photo Album node).
  2. Select the {[%hh_ii:Edit]} Edit | Edit... menu item, the Photo album properties dialog will appear.
  3. Specify thumbnail parameters as needed.
  4. Save changes clicking OK.

However, changing thumbnail parameters itself affects only newly added pictures, and has no effect on existing pictures. In order to have those thumbnais having the same characteristics, you should rebuild them. {[%hh_h2:Rebuilding thumbnails]}

You rebuild thumbnails in PhoA using the File operations wizard:

  1. Run the File operations wizard. You do it using the {[%hh_ii:FileOps]} Tools | File operations... menu item.
  2. On the first page of the wizard select Rebuild thumbnails and click Next.
  3. As long as changing thumbnail parameters should affect all of the photo album thumbnails, choose All pictures in the photo album.
  4. After clicking Next the program will start rebuilding thumbnails for all pictures.
