Browse mode

As was mentioned above, browse mode is intended for photo album creation. This mode is just one the program starts in.

The main program window looks like standard Windows Explorer: the left pane holds folder-groups tree hierarchy while the right one gives us a list of thumbnails for the group selected in left pane.

Right to the left of the root tree node - photo album node you can see dropdown menu button, allowing you to display menu with photo album views commands, as well as for switching between views and photo album groups.
