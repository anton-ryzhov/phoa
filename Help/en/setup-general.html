{[%hh_hdr_en:Program settings :: General]} {[%hh_h2:User interface]}

Interface language

Common program interface language, Russian or English at the moment.

Main program font

Base font used for all PhoA windows.

Visual theme

Allows you to choose a visual theme for toolbars and the status bar.

Tooltip display length

Time to display tooltips in milliseconds.

Recent open files list count

Number of files being remembered in the File menu and Open... toolbutton dropdown list. {[%hh_h2:Clipboard]}

Clipboard copy formats

The set of check boxes allows you to configure formats used for clipboard operations. {[%hh_h2:Toolbars]}

Main toolbar button size

Determines the size of the main (browser) toolbar buttons.

Toolbar dragging style

Determines the toolbar dragging and docking behaviour. {[%hh_h2:Trees/ists]}

Animated expanding/collapsing

If on, expand and collapse operations appear animated.

Panning using mouse wheel click

If on, the mouse wheel click enters autopanning mode.

Center selection in view

If on, the program will try to center the selected item in the list window.

Incremental search on typing

If on, the selection in lists automatically advances to a node when you type its text.

Incremental search cleanup delay

Specifies the time, in milliseconds, during which the sequential keypresses are assumed to be one cycle of incremental search.

Collapse/expand button style

Allows you to choose style of tree buttons (only on non-WindowsXP-systems).

Node checkboxes style

Allows you to choose kind of check images (only on non-WindowsXP-systems).

Multiple selection rectangle style

Allows you to choose style of multiselection rectangle used in lists (only on non-WindowsXP-systems). {[%hh_seealso_en]} Program settings :: Browse mode
