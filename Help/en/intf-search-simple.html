{[%hh_hdr_en:Find pictures :: Simple search]}

Simple search allows you to visually construct the picture search criteria based on picture propertrty values. {[%hh_h2:Controls]}
Reset Deletes all the entered criteria.
Delete Deletes the criterion currently highlighted.
Convert to expression Compiles an expression for expression search from the criteria already entered, and places it to the expression editor.
Search criterion list You enter search criteria in the criterion table using the following data:
  • Picture property - one of the searchable items from picture data, for which the criterion is entered.
  • Condition - one of comparison conditions, list of possible values depends on property data type.
  • Value - a value to compare picture property with.

To add new criterion simply select a property by clicking the dropdown button in the last (empty) table row.

Use arrow keys no navigate through the table. To expand the dropdown list use Alt+Down keys.

Criterion values may be selected from the dropdown history list.

The program select those pictures that conform to all search conditions at once (so conditions are joined using AND operator). {[%hh_seealso_en]} Find pictures
