{[%hh_hdr_en:Find pictures dialog]}

PhoA has a powerful tool for searching pictures. To display Find pictures dialog you select Tools | Find... menu item or press F3 key in the Browse mode. {[%hh_h2:Controls]}
Search criteria list You enter search criteria in the table with the following data:
  • Picture property - one of the searchable item from picture data, for which the criterion is intended.
  • Criterion condition - one of comparison conditions, list of possible values depends on property data type.
  • Criterion value - a value to compare picture property with.

Use arrow keys no navigate through the table. To expand the dropdown list use Alt+Down keys.

Criterion values may be selected from the dropdown history list.

Find Starts picture searching.
Pictures found in search operation (if there are any) fill the Search Results folder that is displayed in the root of the group tree. The Search Results folder implements the same functionality as regular picture group folders except that you cannot add or remove its pictures.
Reset Clears all input fields and reverts all values to their defaults.
{[%hh_h2:Picture properties data types]}

According to the picture data table, PhoA uses the following data types for property values:
Data type Description & criteria
Integer or Ordinal An integer value can be compared for equality, inequality etc.
Filename string Data may be tested for matching the mask. The mask is written by the standard PhoA rules.
String Any literal. Searching string or substring is implemented, case insensitive.
Pixel format Not a searchable type.
Date Criteria are similar to number ones.
Time Criteria are similar to number ones.
String list It is a list of keywords. Keywords may be entered manually or by clicking "..." button in the entry field from the list.

The program select those pictures that conform to all search conditions at once (ie, an AND operator has effect here).

To disable a criterion simply delete its value. {[%hh_seealso_en]} Picture data
