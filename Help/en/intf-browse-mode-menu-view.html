{[%hh_hdr_en:Browse mode menu commands :: View menu]}

This menu controls the program window appearance.
Menu Item Description
Toolbar Controls visibility of the Main toolbar.
Status bar Controls visibility of the Status bar.
Flat mode When Flat mode is active, the thumbnail window shows not only the thumbnails from the group currently selected in the group tree, but also from all its subgroups. Picture reordering by draggin them with mouse is allowed only if the current group doesn't contain subgroups.
{[%hh_ii:RemoveSearchResults]} Remove Search Results Hides the Search Results folder.
Below the photo album view menu items follow.
{[%hh_ii:Folder]} Picture groups Switches group window display to the basic mode, when the group window shows photo album groups (the default selection).
Below follow the photo album view items. When you select an item the corresponding view is activated, and the group tree displays an automatically built view group hierarchy.
{[%hh_ii:NewView]} New view... Opens the Photo album view properties dialog for creating a new view.
{[%hh_ii:Delete]} Delete view Removes the currently active view (in a case a view is active).
{[%hh_ii:Edit]} Edit view... Displays the Property dialog for editing the currently active view (in a case a view is active).
{[%hh_ii:GroupFromView]} Make a group... Opens the Select picture group dialog, allowing you to copy view group hierarchy into a selected photo album picture group.
