{[%hh_hdr_en:Browse mode menu commands :: View menu]}

This menu contains program appearance control commands:

  • Whether toolbar is visible (View | Toolbar menu item).
  • Whether status bar is visible (View | Status bar menu item).
  • {[%hh_ii:RemoveSearchResults]} The View | Remove Search Results menu item hides the Search Results folder.
  • After those the photo album view menu items follow:
    • {[%hh_ii:Folder]} Picture groups item swithes group tree to the main display mode, when the left program window shows photo album groups (default selection).
    • {[%hh_ii:NewView]} New view... menu item opens the Photo album view properties dialog for creating new view.
    • {[%hh_ii:Delete]} Delete view item removes the currently active view (in the case just view is active, not picture groups).
    • {[%hh_ii:Edit]} Edit view... item displays the Property dialog for editing the currently active view (in the case just view is active, not picture groups).
    • {[%hh_ii:GroupFromView]} Make a group... item opens the Select picture group dialog, allowing you to copy view's exact folder hierarchy to a selected photo album picture group.
