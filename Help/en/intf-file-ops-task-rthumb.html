{[%hh_hdr_en:Picture file operations wizard :: Rebuild thumbnails operation]}

You need to rebuild thumbnails each time you change their parameters (which are configured in photo album properties dialog): dimensions, quality or resampling (the latter is adjusted in browse mode settings). In that case you have to build new thumbnails - according to the new parameters. Also this operation is useful when source image files have been changed.

To perform this operation you need to start the Picture file operations wizard and on the Task selection page select Rebuild thumbnails.

The next pages will be:

  • Picture selection
  • Operation process
  • Operation log
