Picture file operations wizard

This wizard performs a number of picture files related operations. It is invoked from the browse mode by selecting Tools | File operations... menu item, or using Ctrl+R shortcut keys.

The wizard is capable of performing the following tasks:

  • Copying the picture files into a specified folder
  • Moving the picture files into a specified folder
  • Deleting unneeded pictures with corresponding files
  • Rebuilding picture thumbnails
  • Repairing links to picture files

Wizard page descriptions:

  1. Task selection
  2. Picture selection
  3. Destination folder selection
  4. Copy/move options (page 1)
  5. Copy/move options (page 2)
  6. CD/DVD creation options
  7. Deletion options
  8. File links repair options
  9. Operation process
  10. Operation log
