{[%hh_hdr_en:Picture file operations wizard :: Copy picture files operation]}

Use this operation to get a copy of picture files in a specified folder. The operation supports many options; in particular, it allows preparing a photo album to write onto a removable medium such as CD, DVD etc.

To perform this operation you need to start the Picture file operations wizard and on the Task selection page select Copy picture files to a folder.

The next pages will be:

  • Picture selection
  • Destination folder selection
  • Copy/move options (page 1)
  • Copy/move options (page 2)
  • CD/DVD creation options
  • Operation process
  • Operation log
