{[%hh_hdr_en:Common tasks :: Viewing pictures]}

PhoA has a built-in image viewer.

The most simple way to view a picture is to make a double-click on it. To quit the view mode and return to the browser press Esc.

You can advance to the next or previous picture using Page Up / Page Down, BkSp / Space keys or mouse wheel. Picture sequence is just the same as for thumbnails in the browse mode. {[%hh_ftr_en]}
