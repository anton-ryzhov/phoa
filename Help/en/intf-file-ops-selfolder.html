Picture file operations wizard :: Destination folder selection

This wizard page serves for selecting an operation's destination folder and is applicable for the following operations:

  • Copying picture files into a specified folder
  • Moving picture files into a specified folder
  • Repairing links to picture files

You have to make a folder selection in the tree, or type its name in the edit field.

Create folder

Creates a new folder inside the folder currently selected, focuses it and starts editing its name.

NB: For the copy files and move files operations you are allowed to specify a new, nonexistent folder name - in that case the folder will be created automatically during the operation process.
