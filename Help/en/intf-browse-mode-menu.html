{[%hh_hdr_en:Browse mode menu commands]} {[%hh_h2:File menu]}

This menu contains commands for photo album file operations:

  • {[%hh_ii:New]} New photo album is created automatically on the program startup or when you select the File | New menu item. It erases all groups, pictures and views while the photo album properties are being reset to the defaults.
  • {[%hh_ii:Open]} To open a photo album file you created before, you select the File | Open... menu item. This command brings up a standard dialog allowing you to select a file (with the .phoa extension).
  • {[%hh_ii:Save]} To save photo album in the file currently open you select the File | Save menu item.
  • {[%hh_ii:SaveAs]} To save it into another file - the File | Save as... menu item.
  • {[%hh_ii:SaveSettings]} The File | Save current settings to file... menu item displays a Save current settings dialog to an .ini file, details.
  • {[%hh_ii:LoadSettings]} The File | Load settings from file... menu item displays a Load program settings dialog from an .ini file, details.
  • {[%hh_ii:Exit]} The File | Exit menu item confirms whether you wish to save changes if there were any (or if it is selected in the Confirmations settings) and exits the program.
  • Below there are MRU (most recently used) files menu items (the count of stored entries is adjusted with Program settings dialog)
{[%hh_h2:Edit menu]}

This menu gathers photo album editing commands:

  • {[%hh_ii:Undo]} Edit | Undo operation menu item rolls back the changes done to the photo album by the indicated (most recent) operation.
  • {[%hh_ii:UndoHistory]} Edit | Undo history submenu contains the changes made list, allowing you to roll back a number of sequential changes at once.
  • {[%hh_ii:Cut]} Edit | Cut menu item removes pictures highlighted and places it into the clipboard (you can add them later to another picture group).
  • {[%hh_ii:Copy]} Similarly, Edit | Copy menu item copies highlighted pictures to the clipboard.
  • {[%hh_ii:Paste]} Edit | Paste item adds pictures (if any) from the clipboard to the current group. Obviously, you can only add new pictures to a group so all duplicates are skipped. If this is specified in the program settings, PhoA notifies you of number of the pictures actually pasted.
  • {[%hh_ii:Edit]} The photo album is the ultimate parent for all the picture groups, and therefore it is the root node of the group tree. The photo album has a text description and it also defines the thumbnail parameters - you can change them in Photo album properties dialog which appears when you select the Edit | Edit... menu item while photo album node is highlighted.
  • {[%hh_ii:Edit]} A picture group can also be renamed by selecting the Edit | Edit... menu item, but the group node must be highlighted in the tree; alternatively, you can click once on a highlighted group name, or press the F2 key. To cancel edit you press the Esc key or the Enter key to apply the new group name.
  • {[%hh_ii:NewGroup]} To add new group you select the Edit | Add group menu item. The group is created as a child to group currently highlighted. When the photo album node is highlighted the group is created in the photo album's root directory. The default group name is "New group", you change it by editing it and pressing Enter.
  • {[%hh_ii:Delete]} A group can be deleted by selecting Edit | Delete menu item while folder tree is active.
  • {[%hh_ii:NewPic]} To add pictures to a group you select Edit | Add picture... menu item which invokes Add pictures wizard window.
  • {[%hh_ii:Sort]} To arrange (sort) the pictures the Edit | Sort pictures... menu item shows the Sort pictures dialog.
  • {[%hh_ii:SelectAll]} The Edit | Select all highlights all the pictures in the current group.
  • {[%hh_ii:SelectNone]} The Edit | Select none removes the highlighting from all the pictures.
  • {[%hh_ii:ViewMode]} To swith to view mode you select the Edit | View menu item or double-click a picture.
{[%hh_h2:View menu]}

This menu contains program appearance control commands:

  • Whether toolbar is visible (View | Toolbar menu item).
  • Whether status bar is visible (View | Status bar menu item).
  • {[%hh_ii:RemoveSearchResults]} The View | Remove Search Results menu item hides the Search Results folder.
  • After those the photo album view menu items follow:
    • {[%hh_ii:Folder]} Picture groups item swithes group tree to the main display mode, when the left program window shows photo album groups (default selection).
    • {[%hh_ii:NewView]} New view... menu item opens the Photo album view properties dialog for creating new view.
    • {[%hh_ii:Delete]} Delete view item removes the currently active view (in the case just view is active, not picture groups).
    • {[%hh_ii:Edit]} Edit view... item displays the Property dialog for editing the currently active view (in the case just view is active, not picture groups).
    • {[%hh_ii:GroupFromView]} Make a group... item opens the Select picture group dialog, allowing you to copy view's exact folder hierarchy to a selected photo album picture group.
{[%hh_h2:Tools menu]}

Optional tool commands are collected here:

  • {[%hh_ii:Props]} The Tools | Program settings... menu item invokes PhoA browse mode settings dialog.
  • {[%hh_ii:Stats]} The Tools | Statistics... menu item computes and displays various photo album summaries.
  • {[%hh_ii:PicOps]} The Tools | Picture operations... menu item allows you to move, copy, crop and intersect pictures with another group using the Picture operations dialog.
  • {[%hh_ii:FileOps]} The Tools | File operations... menu item starts the Picture file operations wizard allowing you to copy, move, delete the picture files, to rebuild the thumbnails or to repair the broken picture file links.
  • {[%hh_ii:Search]} One of the most important PhoA features is the picture search you invoke selecting Tools | Find.... Refer to this chapter for further information.
{[%hh_h2:Help menu]}

The menu contains the following items:

  • {[%hh_ii:About]} The Help | About... item displays version, copyright, update and contact information.
  • {[%hh_ii:Help]} The Help | Help contents item brings up PhoA Help System you are currently watching.
  • The Help | FAQ shows the PhoA FAQ (Frequently Asked Question) section.
  • {[%hh_ii:Globe]} The Help | PhoA Website item starts up the default Internet browser and then opens the program home site.
