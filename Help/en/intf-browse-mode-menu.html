PhoA Help System

Browse mode menu commands

File menu

This menu contains commands for photo album file operations:

  • New photo album is created automatically on the program startup or when you select the File | New menu item. It erases all groups and pictures while photo album properties are being reset to their defaults.
  • To open a photo album file you created before, you select the File | Open... menu item. This command brings up standard dialog that allows you to select a file (wearing the .phoa extension).
  • To save photo album in the currently open file you select File | Save menu item.
  • To save it into another file - the File | Save as... menu item.
  • The File | Exit menu item confirms whether you wish to save changes (if there were any) and exits the program.
  • Below there are MRU (most recently used) files menu items (number of stored entries is adjusted with Program settings dialog)

Edit menu

This menu gathers photo album editing commands:

  • Edit | Undo operation menu item rolls back changes done to the photo album by signed (most recent) operation.
  • Edit | Undo history submenu item contains the changes made list, allowing you to roll back a number of sequential changes at once.
  • Edit | Cut menu item removes pictures highlighted and places it into the clipboard (you can add them later to another picture group).
  • Similarly, Edit | Copy menu item copies highlighted pictures to the clipboard.
  • Edit | Paste item adds pictures (if any) from the clipboard to the current group. Obviously, you can only add new pictures to a group, all duplicates are skipped. If this specified in the program settings, PhoA notifies you of number of pictures actually pasted.
  • The Photo Album is the ultimate parent for all the picture groups, and therefore it is the root node of the group tree. The photo album can hold a text description and it also defines thumbnail parameters - you can change them in Photo album properties dialog, which appears when you select the Edit | Edit... menu item while photo album node is highlighted.
  • A picture group can be renamed also by selecting the Edit | Edit... menu item, but the group node must be highlighted in the tree; alternatively, you can click once on a highlighted group name, or press F2 key. To cancel edit you press Esc key, to confirm new group name - Enter key.
  • To add new group you select the Edit | Add group menu item. The group is created as a child to group currently highlighted. When the photo album node is highlighted the group is created in the photo album's root directory. The default group name is "New group", you change it by editing it and pressing Enter.
  • A group can be deleted by selecting Edit | Delete menu item while folder tree is active.
  • To add pictures to a group you select Edit | Add picture... menu item which invokes Add pictures wizard window.
  • To arrange (sort) pictures the Edit | Sort pictures... menu item brings up Sort pictures dialog.
  • The Edit | Select all highlights all the pictures in the current group.
  • The Edit | Select none removes highlight from all the pictures.
  • To swith to view mode you select the Edit | View menu item or double-click a picture.

View menu

This menu controls program appearance:

  • Whether toolbar is visible (View | Toolbar menu item) and
  • If status bar is visible (View | Status bar). The program saves visibility of these items and restores it next time you run it.
  • After them photo album views menu items follow:
    • Picture groups item swithes group tree to the main display mode, when the left program window shows photo album groups (default selection).
    • New view... menu item brings up Photo album view properties dialog for creating new view.
    • Delete view item removes the currently active view (in the case just view is active, not picture groups).
    • Edit view... item displays Property dialog for editing the currently active view (in the case just view is active, not picture groups).
    • Make a group... item opens Select picture group dialog, allowing you to copy view's exact folder hierarchy to a selected photo album picture group.

Tools menu

Optional tool commands are collected here:

  • The Tools | Program settings... menu item invokes PhoA browse mode settings dialog.
  • The Tools | Statistics... menu item counts and displays various photo album summaries.
  • The Tools | Rebuild thumbnails... menu item brings up Rebuild thumbnails dialog allowing you to recreate thumbnails for selected pictures, for all pictures in the current group or for the whole photo album. This operation is necessary every time you alter thumbanil parameters in the Photo album properties dialog.
  • The Tools | Picture operations... menu item allows you to move, copy, crop and intersect pictures to another group with the Picture operations dialog.
  • One of the most important PhoA features is picture search you invoke selecting Tools | Find.... Refer to this chapter for further information.

Help menu

The menu contains three items:

  • The Help | About... item displays version, copyright, update and contact information.
  • The Help | Help contents item brings up PhoA Help System you are currently watching.
  • The Help | PhoA Website item starts up the default Internet browser and opens the program home site in it.

To the left of the photo album node in the folder tree dropdown-menu button is placed, which brings up view menu.
