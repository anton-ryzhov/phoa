{[%hh_hdr_en:Sort pictures dialog]}

This function allows you to reorder pictures in the current group, or in all photo album groups based on their data.

If there's no current group, you may only sort the whole photo album. {[%hh_h2:Controls]}
Sort criteria list Sort order is determined by sort criteria, number of criteria may be arbitrary.
Each criterion represents a picture property and a sort direction (ascending / descending). Criteria are being added automatically when you select a property to sort by.
To display property list you click a dropdown-button in the first column or Space key when the first column has focus.
To toggle sort direction you click direction indicator with mouse or press Space key when the second column has focus.
You also can reorder items in the list by dragging them with the mouse or using keys Ctrl+Up and Ctrl+Down.
Reset Reverts all sort criteria to their defaults.
