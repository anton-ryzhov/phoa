{[%hh_hdr_en:Keyboard and mouse controls]}

It is possible in PhoA to perform almost any operation by a number of ways: selecting an item in the main menu, clicking a toolbar button (only in the browse mode), by pressing a keyboard shortcut, selecting an item from popup menu, or, finally, with the mouse. Below are keyboard shortcuts and mouse control for the two major modes:

  • Browse mode:
    • Keyboard
    • Mouse
  • View mode:
    • Keyboard
    • Mouse
