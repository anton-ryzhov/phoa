Program settings

This dialog allows you to alter various program settings. To display it, select Tools | Program settings... menu item, or press F4 shortcut key.

The window contains a number of pages switchable with the buttons at the left, or by using Ctrl+Digit shortcut keys:
Page Shortcut Purpose
General Ctrl+1 General program settings
Browse mode Ctrl+2 Browse mode related settings
View mode Ctrl+3 View mode related settings
Dialogs Ctrl+4 Dialod windows and Wizard settings
Tools Ctrl+5 Tool editor
