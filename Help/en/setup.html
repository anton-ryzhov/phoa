{[%hh_hdr_en:Program settings]}

This dialog allows you to alter various program settings. To display it, select Tools | Program settings... menu item, or press F4 shortcut key.

The window contains a number of pages switchable with the buttons at the left, or by using Ctrl+Digit shortcut keys:
Page Shortcut Purpose
{[%hh_ii:Props]} General Ctrl+1 General program settings
{[%hh_ii:Folder]} Browse mode Ctrl+2 Browse mode related settings
{[%hh_ii:ViewMode]} View mode Ctrl+3 View mode related settings
{[%hh_ii:Dialog]} Dialogs Ctrl+4 Dialog windows and Wizard settings
{[%hh_ii:Tool]} Tools Ctrl+5 Tool editor
