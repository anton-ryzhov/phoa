{[%hh_hdr_en:Picture file operations wizard :: CD/DVD creation options]}

This wizard page allows specifying extended options for CD, DVD and other removable media creation for the following file operations:

  • Copying picture files into a specified folder
  • Moving picture files into a specified folder
{[%hh_h2:Controls]}
Create a photo album file with selected pictures to If turned on, the program creates a new photo album based on selected groups and pictures, and places it in the file specified in the next field.
Photo album file name Specifies the name of the file to save photo album to (with .phoa extension). The currently open photo album file name is used by default.
Photo album description Allows you to specify the description for the newly created photo album. The currently open photo album description is used by default.
Also include existing views If on, all the views from the current photo album will be added to the photo album being created.
Copy the program itself to the destination folder If on, the main program executable (phoa.exe) will also be copied into the destination folder. Since (as it's well known) the program doesn't require to be installed to be run on a machine, it can be executed from any media, including removable ones.
Copy the current program settings to phoa.ini If on, the current program settings will be saved to phoa.ini file in the destination fodler during copy/move operation. This allows running the program from a removable media keeping all the current settings on any machine (but only if setting autoloading from phoa.ini is not disabled in that machine's settings.)
Create autorun.inf file This option allows to automatically generate autorun.inf file placed in the destination folder. In this case inserting CD-ROM to a computer will cause automatic PhoA startup, and then the program will load the photo album specified above.
NB: Autorun is functional unless this feature is disabled on a machine.
Label the media as Specifies media name that is displayed in My computer. The wizard uses Photo album, or current view name by default.
NB:
  • This field is optional, and can be left empty. My computer will display the media label if this is the case.
  • Media label is displayed unless autorun feature is disabled on a computer.

ATTENTION: all the contents of the folder designated as destionation folder must reside in the root directory of the written disk! {[%hh_seealso_en]} Picture file operations wizard
