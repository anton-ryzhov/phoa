{[%hh_hdr_en:Group properties dialog]}

This dialog allows for viewing or editing picture group properties.

To display it use the Edit | Edit... menu item when the group tree is focused and one of the groups is highlighted (but not the photo album, a view or Search results group).

ID

A read-only field. Displays an unique numeric group identifier used internally by the program and similar to a picture identifier. The group is being assigned the identifier automatically when it is created or loaded from a photo album had been created by an older program version (prior to 1.1.5).

Text

Specifies the text (or the name) of the group that is displayed in the tree.

Description

Here you can enter some additional informaton about the pictures contained in the group. The description can be displayed in the tooltips if configured in program settings.

NB: A group can be renamed alternatively by inplace editing the group name. You enter the rename mode with the F2 key or by left mouse click on already highlighted group name in the browse mode.
