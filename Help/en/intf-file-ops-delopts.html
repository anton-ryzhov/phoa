{[%hh_hdr_en:Picture file operations wizard :: Deletion options]}

This wizard page specifies picture file deletion operation options.

Delete to Recycle Bin

  • If this option is off, all the files are being wiped off and cannot be restored.
  • If the option is on, the files are being moved into the Recycle Bin rather than deleted (if this is possible on the given machine).
NB: All the operations being performed (succeeded or failed) log their results which can be inspected later.
