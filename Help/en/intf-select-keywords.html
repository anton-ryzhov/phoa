PhoA Help System

Select keywords dialog

You enter this dialog by clicking on the keyword list button in the Find pictures. The program displays the complete list of keywords used in the photo album - sorted in ascending order.

You select ones you wish to search with.

Reset

Clears all keyword checkmarks.

NB: if a keyword contains spaces or commas it becomes double-quoted because these characters are delimiters.
