{[%hh_hdr_en:Find pictures dialog]}

PhoA has a powerful tool for searching pictures. To display Find pictures dialog you select Tools | Find... menu item or press F3 key in the Browse mode. {[%hh_h2:Controls]}
Perform search Determines the scope of the search:
  • In the photo album - all the available pictures are searched.
  • In the current group - perform search only among pictures of the group currently selected (disabled if no group selected).
  • In the search results - search the results of previous search (enabled only if there is the Search results folder).
Search criteria PhoA supports the following two search methods:
  • Simple search allows you to specify some picture properties and to check conformity of its values to some conditions.
  • Expressions search allows for writing an arbitrary filter expression, which determines the searching criteria.
Find Starts picture searching. Pictures found in search operation (if any) are put into the Search Results folder which is displayed in the root of the group tree. The Search Results folder implements the same functionality as regular picture group folders except for add and remove pictures.
