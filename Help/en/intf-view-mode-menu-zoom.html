{[%hh_hdr_en:View mode :: Zoom menu]}

This menu contains image scaling commands.
Menu Item Description
{[%hh_ii:ZoomIn]} Zoom in Enlarges the image by a value specified in the view mode settings. You can also perform the command with Numpad + or Ctrl+Wheel up.
{[%hh_ii:ZoomOut]} Zoom out Shrink the image by a value specified in the view mode settings. You can also perform the command with Numpad - or Ctrl+Wheel down.
{[%hh_ii:ZoomActual]} Zoom actual Sets the scale to 100% (or 1:1), when one image pixel matches one screen pixel. You can also perform the command with Numpad /.
{[%hh_ii:ZoomFit]} Zoom best fit Sets the scale factor so that image becomes fully visible in the window. You can also perform the command with Numpad *.

NB:
  • Maximum image scale is 8:1.
  • Minimum image scale is 1:10.
