{[%hh_hdr_en:Picture file operations wizard :: Delete picture files operation]}

Use this operation to delete unneeded or broken picture files (obviously, with the corresponding pictures; otherwise those pictures would link to nonexistant files). The wizard allows complete picture file deletion as well as moving them to the Recycle Bin instead.

To perform this operation you need to start the Picture file operations wizard and on the Task selection page select Delete pictures and files.

The next pages will be:

  • Picture selection
  • Deletion options
  • Operation process
  • Operation log
