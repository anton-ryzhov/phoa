Rebuild thumbnails dialog

Thumbnails must be rebuilt every time you alter their parameters (those defined in photo album properties): dimensions and/or quality. In such case it is necessary to recreate new thumbnails that are appropriate for new settings. Also this function is useful when source picture files change.

Rebuild thumbnails function allows you to select one of the following modes:

  • for all selected pictures - thumbnails are rebuilt only for pictures being highlighted in the right pane of PhoA window;
  • for all pictures in the current group - thumbnails are rebuilt for pictures that are in the group highlighted in the PhoA group tree;
  • for all photo album pictures - all thumbnails are rebuilt.

During picture processing PhoA displays progress indicator as well as file name of picture being processed; you can interrupt rebuilding at any moment.
