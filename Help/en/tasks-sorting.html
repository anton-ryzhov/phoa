{[%hh_hdr_en:Common tasks :: Sorting pictures]}

Sorting of pictures with reference to PhoA means ordering them by certain criteria.

NB: Each group has its own picture links. Therefore you can have, say, two groups containing the same set of pictures but ordered differently.

The program allows you to order picture by any properties available. The most common case for photos is a chronological sequence, that is by date and time.

You can sort pictures either in the current group, or in all photo album groups at once. In order to accomplish this:

  1. If you want to sort only one group, select that group in the tree.
  2. Select the {[%hh_ii:Sort]} Edit | Sort pictures... menu item. This will show the Sort pictures dialog.
  3. Choose an appropriate option at the Where to sort pictures switch.
  4. Specify sorting criteria. The dropdown-arrow button allows you to choose a sort property, or to delete a row. In order to add a new property, choose it in the last empty row of the table.
  5. Click OK.
NB: Just as above you can sort the search results, but in order to perform this you should highlight the Search results folder in the group tree and explicitly choose In the current group.
