{[%hh_hdr_en:Edit pictures dialog :: File properties]}

The page displays file list of pictures that are selected. If you expand the file name you can see that file's properties (type, size, creation, modification and accessed time etc).

By clicking the right mouse button you can invoke standard Explorer context menu for a file.

NB: You may opt to expand all the properties in dialog settings.
