{[%hh_hdr_en:View mode :: View menu]}

This menu controls the viewing window appearance.
Menu Item Description
Main Menu Toggles the view window Main Menu bar.
Main Toolbar Toggles the Main Toolbar.
Transformations Toolbar Toggles the Transformations Toolbar.
Slide Show Toolbar Toggles the Slide Show Toolbar.
{[%hh_ii:FullScreen]} Fullscreen mode Switches between fullscreen and window viewing modes. You can also perform the switching by using the F key or with middle mouse button (wheel) click.
{[%hh_ii:Refresh]} Reload Reloads the picture currently displayed, from the file. This may appear useful to reload a picture changed externally.
{[%hh_ii:Info]} Show info Shows or hides the picture information bar.
{[%hh_ii:InfoRelocate]} Relocate info bar Enters the info bar positioning mode, when the info bar has eight drag handles on its bounds and you are able to resize it or change its position by dragging it with the mouse. To finish positioning, select this item again or click somewhere outside the bar.
