{[%hh_hdr_en:Browse mode menu commands :: Tools menu]}

Additional tool commands are collected in this menu.
Menu Item Description
{[%hh_ii:Props]} Program settings... Invokes the PhoA settings dialog.
{[%hh_ii:Stats]} Statistics... Computes and displays various photo album summaries.
{[%hh_ii:PicOps]} Picture operations... Allows you to move, copy, crop and intersect pictures with another group using the Picture operations dialog.
{[%hh_ii:FileOps]} File operations... Starts the Picture file operations wizard allowing you to copy, move, delete the picture files, to rebuild the thumbnails or repair the broken picture file links.
{[%hh_ii:Search]} Find... Displays the Find pictures dialog. Refer to this chapter for further information.
