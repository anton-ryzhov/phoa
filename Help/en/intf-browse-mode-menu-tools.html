{[%hh_hdr_en:Browse mode menu commands :: Tools menu]}

Optional tool commands are collected here:

  • {[%hh_ii:Props]} The Tools | Program settings... menu item invokes PhoA browse mode settings dialog.
  • {[%hh_ii:Stats]} The Tools | Statistics... menu item computes and displays various photo album summaries.
  • {[%hh_ii:PicOps]} The Tools | Picture operations... menu item allows you to move, copy, crop and intersect pictures with another group using the Picture operations dialog.
  • {[%hh_ii:FileOps]} The Tools | File operations... menu item starts the Picture file operations wizard allowing you to copy, move, delete the picture files, to rebuild the thumbnails or to repair the broken picture file links.
  • {[%hh_ii:Search]} One of the most important PhoA features is the picture search you invoke selecting Tools | Find.... Refer to this chapter for further information.
