{[%hh_hdr_en:Find pictures :: Data types]}

According to the picture data chart, PhoA uses the following data types for property values:
Data type Description & criteria
Integer or Ordinal An integer value can be compared for equality, inequality etc.
String Any literal. Searching string or substring is implemented, case insensitive.
Pixel format Not a searchable type.
Date Criteria are similar to number ones.
Time Criteria are similar to number ones.
Size (Width x Height) Not a searchable type.
String list It is a list of keywords. Keywords may be entered manually or by clicking "..." button in the entry field from the list.
Picture rotation Not a searchable type.
Picture flips Not a searchable type.
