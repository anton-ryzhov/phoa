{[%hh_hdr_en:Program settings :: Dialogs]}
Show splash screen at startup If on, a splash screen is shown at program startup.
Animate startup splash fadeout If on, the startup splash screen will fadeout slowly after the program is idle (only under Windows 2000 and later).
Animate About dialog fadeout If on, the About dialog will fadeout slowly when being closed (only under Windows 2000 and later).
Confirmations
Before group deletion If on, the program confirms before a group is deleted.
Before picture deletion If on, the program confirms before selected pictures are deleted.
Before view deletion If on, the program confirms before a view is deleted.
Before saving file of an older version If on, when saving file of a version older than the current, the program warns you about possible data loss.
Before exit (when photo album is not modified) If on, the program confirms before exiting even if the photo album isn't modified (if the photo album is modified the program always confirms exit).
Notifications
After copying pictures using Drag'n'Drop If on, after copying pictures the program notifies you about number of pictures actually copied.
After moving pictures using Drag'n'Drop If on, after copying pictures the program notifies you about number of pictures actually moved.
After pasting pictures from the clipboard If on, after pasting pictures the program notifies you about number of pictures actually pasted.
Add pictures wizard
Show hidden files and folders Controls displaying of files and folders having Hidden attribute.
Extract file icons If on, the program will extract system file icons during file list compilation (those icons are displayed in the file list on the check files page). Enabling the option may slow down file list compilation, especially for large number of files.
Always skip 'Check files' page If on, after selecting files/folders in the Add pictures wizard PhoA will start picture addition process immediately.
Show log page only if errors occured If on, addition log will be displayed only if there were errors, otherwise the wizard window will close right after addition process is finished. If off, the log will be displayed each time upon completion.
Automatically fill picture date from: Adjusts properties from which to autofill picture date for pictures being added.
Overwrite date if already specified Tells the program to autofill picture date (overwrite date) even if the date is already entered.
Automatically fill picture time from: Adjusts properties from which to autofill picture time for pictures being added.
Overwrite time if already specified Tells the program to autofill picture time (overwrite time) even if the time is already entered.
Automatically fill picture transformations from metadata Will apply the required transformations to the picture during the autofilling properties for pictures being added. The need for transformations is determined from the Orientation metatag.
Picture properties dialog
Page displayed by default Specifies the default page to display when you open the Picture properties dialog.
Expand file properties on display If on, the file properties on the property page fully expand when displayed.
Expand image metadata on display If on, the image metadata on the metadata page fully expand when displayed.
Picture file operations Wizard
Confirm Copy files operation If on, the program will confirm proceeding with the Copy files operation.
Confirm Move files operation If on, the program will confirm proceeding with the Move files operation.
Confirm Delete files operation If on, the program will confirm proceeding with the Delete files operation.
Confirm Rebuild thumbnails operation If on, the program will confirm proceeding with the Rebuild thumbnails operation.
Confirm Repair file links operation If on, the program will confirm proceeding with the Repair file links operation.
Show log page only if errors occured If on, file operation log will be displayed only if there were errors, otherwise the Picture file operations wizard window will close once the operation is complete. If off, the log will be displayed each time upon the completion.
