{[%hh_hdr_en:Picture file operations wizard :: Repair picture file links operation]}

This operation allows you to repair picture file links broken by some reason by searching the corresponding files in the specified folder.

To perform this operation you need to start the Picture file operations wizard and on the Task selection page select Repair picture file links.

The next pages will be:

  • Picture selection
  • Destination folder selection
  • File links repair options
  • Operation process
  • Operation log
