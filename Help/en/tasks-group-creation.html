{[%hh_hdr_en:Common tasks :: Group creation]} {[%hh_h2:Why create groups?]}

Picture group in PhoA help you to arrange photos by category. You can create a hierarchical structure of picture groups like you do it in the Explorer.

To create a group, select a parent group for it from the tree and select the {[%hh_ii:NewGroup]} Edit | Add group menu item. A new group will be created inside the selected one, and the program will enter group name editing mode. Type in a name for the new group (for example, "My birthday") and click Enter. {[%hh_h2:Reordering groups]}

You can change the order in which groups appear in the tree as well as move one group inside another one by dragging it with the mouse.

  • In order to place a group inside another group you should drag it directly onto the latter group's node.
  • To place a group near another group drag it to a blank space to the right of the groups. This will display an insertion line shaped accordingly to mouse cursor position.
{[%hh_h2:Adding pictures to a group]}

You can add pictures to the group currently highlighted using the Add pictures wizard. {[%hh_ftr_en]}
