Edit pictures dialog :: Metadata

This page displays metadata contained in picture files. Metadata are arranged as tree, you can view them by expanding a file name.

If there are no metadata in the file, the corresponding message appears to the right of its name.

The bottom panel, Description, displays additional information for the selected metatag. This panel is visible by default, but can be closed by clicking the X button on it. To display the panel again, select the corresponding item from the context menu.

NB:
  • You may opt to expand all the metadata in dialog settings.
  • Metadata may be used for autofilling the picture properties when adding pictures to the photo album. This is adjusted also in dialog settings.
