{[%hh_hdr_en:Program settings :: Browse mode]} {[%hh_h2:Picture group trees]}

Tooltips

Configures usage and appearance of tooltips in the trees displaying picture groups:

  • None - tooltips disabled.
  • Display if item is truncated at window edge - highlights the item not fitting into the window so that you are able to view it whole.
  • Display with selected properties - the tooltips contain group properties you select (see below). Additionally, the photo album description is displayed for the photo album node when you hover it.

Display in tooltips

The list of group properties to select from. Applicable only if tooltips use the Display with selected properties mode. {[%hh_h2:Thumbnail view window]}

Background color

Determines the thumbnail pane background color.

Thumbnail background color

Determines the background color for single thumbnails.

Thumbnail font color

Determines the font color for thumbnail labels.

Drag'n'Drop

Enables or disables drag'n'drop operations with the thumbnails.

Show thumbnail tooltips

This function enables pop-up tooltips for thumbnails when mouse stays still over a thumbnail for a time. These tooltips display picture properties (see below). Being generally useful, this function may irritate someone persons.

Display in tooltips

The list of properties from which you can choose those to display in thumbnail tooltips.

Picture details displayed on the thumbnails

By using these options you can select whether to display, and which picture properties in each thumbnail corner.

Thick thumbnail border

When enabled, thumbnail boundaries are being drawn more plain.

Cache thumbnails while viewing

Since thumbnails are stored in the JPEG format, they must be decoded to be drawn, and it may take sensible time to repaint them each time they need to (particularly when displaying many thumbnails at one time while PC is not so powerful). To amend this caching function enables storing decode thumbnails in the special cache (after they had been painted once). Although it takes a bit more memory.

Thumbnail cache size

Maximum number of thumbnails in the cache.

Resampling filter

Determines interpolation filter used for resampling source images when creating thumbnails.

Max. undo count

Determines size of undo buffer used to roll back changes done to the photo album. You can undo no more than specified number of sequential changes (when undo buffer gets full, the least recent changes' data are erased). Large buffer requested may result in large need of room (memory) to store it (the most memory-consuming operations are picture deleting and editing). {[%hh_seealso_en]} Program settings :: General
