{[%hh_hdr_en:Common tasks :: Slide show]}

The program is capable of working in the Slide Show mode. This is a mode when images change each other automatically after certain intervals.

You start slide show from the view mode by pressing the Pause key, using the {[%hh_ii:SlideShow]} Tools | Slide show | Slide show menu item, or with the corresponding toolbar button.

A required flip delay is configured in the program settings, or by using the toolbar. In the latter case the delay value won't be saved when you exit the view mode (in other words, it affects the current view session only). {[%hh_ftr_en]}
