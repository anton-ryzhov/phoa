{[%hh_hdr_en:Picture file operations wizard :: Operation process]}

This wizard page shows the operation process state:

  • Thumbnail of the picture being processed
  • Progress bar
  • Count of pictures successfully processed
  • Processing fail count
{[%hh_h2:Controls]}
Interrupt/Continue This button allows interruption of current operation; in the case clicking Next > button advances to the operation log page. The process has been interrupted can be resumed by a second click on the button.
