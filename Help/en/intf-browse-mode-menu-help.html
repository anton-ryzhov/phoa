{[%hh_hdr_en:Browse mode menu commands :: Help menu]}

The menu contains PhoA Help System items, and open program web site commands:
Menu Item Description
{[%hh_ii:About]} About... Displays version, copyright, update and contact information.
{[%hh_ii:Help]} Help contents Invokes the PhoA Help System you are currently watching.
FAQ Shows the PhoA FAQ (Frequently Asked Question) section.
{[%hh_ii:Asterisk]} Internet | Check for updates Starts the default Internet browser and opens the Program update check page.
{[%hh_ii:Globe]} Internet | PhoA home site Starts the default Internet browser and opens the PhoA home page.
Internet | Product support Starts the default Internet browser and opens the Program support page.
{[%hh_ii:DKSoftware]} Internet | DK Software Website Starts the default Internet browser and opens the Program vendor page.
