{[%hh_hdr_en:Edit pictures dialog]}

This dialog is intended for modifying selected pictures. To display it, select one or more thumbnails and click {[%hh_ii:Edit]} Edit | Edit... menu item or press Alt+Enter keys provided the thumbnail pane is focused.

The window contains several pages switched by the toolbar at its left edge, or by pressing Ctrl+Digit keys:
Page Shortcut Purpose
{[%hh_ii:Props]} File properties Ctrl+1 Viewing general picture files properties
{[%hh_ii:Metadata]} Metadata Ctrl+2 Viewing metadata contained in the picture files
{[%hh_ii:ViewMode]} View Ctrl+3 Viewing the pictures themselves at selected scale and applying the transformations
{[%hh_ii:Edit]} Data Ctrl+4 Viewing or editing the picture data
{[%hh_ii:Keyword]} Keywords Ctrl+5 Viewing or editing the picture keywords
{[%hh_ii:Folder]} Groups Ctrl+6 Determining or changing the pictures' belongings to the groups
{[%hh_undosup_en]} {[%hh_seealso_en]} Browse mode Working in the browse mode
