Edit pictures dialog

This dialog is intended for modifying selected pictures. To display it, select one or more thumbnails and click Edit | Edit... menu item or press Alt+Enter keys provided the thumbnail pane is focused.

The window contains several pages switched by the toolbar at its left edge, or by pressing Ctrl+Digit keys:
Page Shortcut Purpose
File properties Ctrl+1 Viewing general picture files properties
Metadata Ctrl+2 Viewing metadata contained in the picture files
View Ctrl+3 Viewing the pictures themselves at selected scale
Data Ctrl+4 Viewing or editing the picture data
Keywords Ctrl+5 Viewing or editing the picture keywords
Groups Ctrl+6 Determining or changing the pictures' belongings to the groups

This operation supports undo
