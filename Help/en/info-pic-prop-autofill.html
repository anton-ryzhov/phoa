{[%hh_hdr_en:Autofilling the picture properties]}

PhoA is capable of automated filling some picture properties. At the moment there are the following autofill capabilities:
Property Autofill possibility
Date
  • From the Original date & time EXIF tag
  • From the Dititized date & time EXIF tag
  • From the Date & time EXIF tag
  • Extraction from the picture file name - if it has yyyy-mm-dd hh-mm-ss.* format
  • From file creation date/time
  • From file modification date/time
Time
  • From the Original date & time EXIF tag
  • From the Dititized date & time EXIF tag
  • From the Date & time EXIF tag
  • Extraction from the picture file name - if it has yyyy-mm-dd hh-mm-ss.* format
  • From file creation date/time
  • From file modification date/time
Transformations
(rotation, flips)
  • From the Orientation EXIF tag

The program will try to fill a particular picture property, iterating through items enabled in dialog settings in the order they are listed. If a property was filled successfully the program stops iterating.

Moreover Overwrite if already specified option also affects autofill. {[%hh_seealso_en]} Program settings :: Dialogs
