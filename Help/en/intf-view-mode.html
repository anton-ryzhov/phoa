{[%hh_hdr_en:View mode]}

This mode allows you to view single pictures one by one - either in a separate window or in the full-screen mode. And this is probably what you create the photo album for.

In the view mode you can do the following:

  • scale (zoom in and out) the viewed picture;
  • scroll the picture with the mouse or keyboard in a case it isn't fully visible in the window;
  • advance to the next, the previous, the first or the last picture in the current view list;
  • switch between window and fullscreen viewing modes;
  • edit the current picture properties;
  • turning on and off the Slide Show mode;
  • open program preferences & so on.

These functions are available through the Main Menu, the Toolbar buttons, by using the context menu that appears when you right-click the window, or using the keyboard and mouse controls. {[%hh_seealso_en]} Working in the view mode
