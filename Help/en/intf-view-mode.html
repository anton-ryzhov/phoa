View mode

This mode allows you to view singe pictures one by one - in the separate window or in fullscreen mode. And this is generally what you create photo album for.

In the view mode you can do the following:

  • scale (zoom in and out) picture;
  • scale picture to best fit the window;
  • cancel scaling (view picture in its actual size);
  • scroll picture with mouse or keyboard in case it doesn't fit the window;
  • advance to next, previous, the first or the last picture in the current group;
  • switch between window and fullscreen viewing mode;
  • reload (re-read) picture file;
  • edit current picture;
  • turning on and off slideshow mode;
  • open program preferences.

These function are available through toolbar buttons, by context menu that appears when you right-click the window, or by keyboard and mouse controls.
