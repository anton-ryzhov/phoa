PhoA Help System

Photo Album properties dialog

This dialog allows you to define photo album thumbnails' parameters - their dimensions (width and height in pixels) and quality. Default thumbnail dimensions are 150x150 pixels, quality is 40% max. The greater thumbnail dimensions and the higher quality mean the greater photo album file (*.phoa) size, and vice versa. Too low thumnail quality may make it difficult to recognize picture it represents (colors appear distorted).

You can also enter photo album text description here; the description is stored along with the photo album.

NB: Description length is not limited but it's not recommended to make descriptions over 80 printed pages in length because they are difficult to read.

This operation supports undo
