{[%hh_hdr_en:Edit pictures dialog :: View]}

Here you can view any of the selected picture at the selected scale. Picture file (if there are more than one picture selected) is chosen from the dropdown list. {[%hh_h2:Controls]}
{[%hh_ii:ZoomIn]} Zoom in Enlarges picture for value denoted in View settings.
{[%hh_ii:ZoomOut]} Zoom out Shrinks picture for value denoted in View settings.
{[%hh_ii:ZoomActual]} Zoom actual Set the scale to 1:1.
{[%hh_ii:ZoomFit]} Fit window Scales picture so that it fits the window. This is the default.
{[%hh_ii:Rotate0]} No rotation Don't apply rotation to the image when it is displayed.
{[%hh_ii:Rotate90]} Rotate CW by 90° Apply a 90° clockwise rotation to the image when it is displayed.
{[%hh_ii:Rotate180]} Rotate by 180° Apply a 180° rotation to the image when it is displayed.
{[%hh_ii:Rotate270]} Rotate CCW by 90° Apply a 90° counter-clockwise rotation (or 270° clockwise rotation) to the image when it is displayed.
{[%hh_ii:FlipHorz]} Flip horizontally Apply a horizontal flip to the image when it is displayed.
{[%hh_ii:FlipVert]} Flip vertically Apply a vertical flip to the image when it is displayed.

NB:
  • These commands are also available from the popup menu appearing on right mouse click.
  • Ctrl+Right click invokes shell context menu (Explorer menu) for the file currently being displayed.
  • Transformation commands are applied to all the selected pictures. If a transformation property (eg. rotation) among the selected pictures, the corresponding transformation buttons (rotation buttons in our case) will be in the unpressed state. If you now decide to apply a rotation the rotation will be the same for all the selected pictures.

Picture panning (in the case the picture is larger than the window) is performed by dragging it with the mouse. Additionally you can use the mouse wheel to zoom the picture.

NB: To speed up picture displaying always Nearest neighbor is used for resampling.
