PhoA Help System

Edit pictures dialog :: View

Here you can view any of the selected picture at the selected scale. Picture file (if there are more than one picture selected) is chosen from the dropdown list.

Zoom in

Enlarges picture for value denoted in View settings.

Zoom out

Shrinks picture for value denoted in View settings.

Zoom actual

Set the scale to 1:1.

Fit window

Scales picture so that it fits the window. This is the default.

NB: these command are also available from the popup menu appearing on right mouse click.

Picture panning (in the case the picture is larger than the window) is performed by dragging it with the mouse. Additionally you can use the mouse wheel to zoom the picture.

NB: To speed up picture displaying always Nearest neighbor is used for resampling.
