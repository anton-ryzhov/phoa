{[%hh_hdr_en:Picture filter expression editor]}

Expression editor is used to create picture filter expressions in PhoA: for Expression search and in View properties.

Detailed description of picture filter expression language) is given here. {[%hh_h2:Controls]}
{[%hh_ii:New]} New Clears the expression editor contents.
{[%hh_ii:Open]} Open... Displays the standard File open dialog which allows you to load text of expression from a file saved before. The dropdown arrow button displays the MRU (most recently used) file list.
{[%hh_ii:SaveAs]} Save as... Displays the standard File save dialog allowing to select a file to write the current expression to.
{[%hh_ii:Something]} Insert property Displays a menu containing the list of available picture properties. When you select an item, property text is inserted at the current caret position in the expression editor.
{[%hh_ii:Something]} Insert operator Displays a menu containing the list of available operators. When you select an item, operator text is inserted at the current caret position in the expression editor.
{[%hh_ii:Cut]} Cut Cuts text currently selected in the expression editor, to the clipboard.
{[%hh_ii:Copy]} Copy Copies text currently selected in the expression editor, to the clipboard.
{[%hh_ii:Paste]} Paste Pastes text from the clipboard into the expression editor.
{[%hh_ii:Undo]} Undo Rolls back the last change made in the expression editor.
{[%hh_ii:Redo]} Redo Redoes the last misundone change.
{[%hh_ii:OK]} Syntax check Performs the syntax check of the entered expression and displays a message saying about check results.

The same commands are available in the expression editor context menu displayed by a mouse right click or using Shift+F10 keys. {[%hh_seealso_en]} Find pictures
