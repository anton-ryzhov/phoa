Program settings :: Tools

Tools page of Program settings dialog is intended for configuring the set of available tools.

The order of tools in the list determines their order in menus.

You can reorder tools by dragging them with the mouse, using popup menu commands, or with Ctrl+Up, Ctrl+Down keys.

To display tool's menu click the item button, right click the item or press Space key.

To delete a tool select Delete from the tool menu or press Del key.

To view or edit tool properties use Edit... tool menu item, press Alt+Enter shortcut keys or double click the item. This invokes Tool properties dialog.

To add new tool entry select Edit... tool menu item for the last, empty list entry.

Column selection

It's up to you to specify which columns are to be displayed in the tools list. You achieve this by invoking the header popup menu, accessible with the right mouse click on the list header. Additionally, you can reorder column by dragging them with the mouse.
