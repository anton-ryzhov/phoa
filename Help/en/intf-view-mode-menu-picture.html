{[%hh_hdr_en:View mode :: Picture menu]}

This menu contains picture advancement and property editing commands as well as command for exiting the View mode.
Menu Item Description
{[%hh_ii:First]} First picture Jumps to the first picture in the view list.
{[%hh_ii:Left]} Previous picture Reverts to the previous picture.
{[%hh_ii:Right]} Next picture Advances to the next picture.
{[%hh_ii:Last]} Last picture Jumps to the last picture in the view list.
{[%hh_ii:Edit]} Edit... Opens the Picture properties dialog for the current picture, allowing you to edit the picture details.
{[%hh_ii:Exit]} Close Ends the viewing session and returns to the Browse mode.
