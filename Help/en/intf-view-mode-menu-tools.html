{[%hh_hdr_en:View mode :: Tools menu]}

This menu contains commands for invoking some tool functions.
Menu Item Description
{[%hh_ii:SlideShow]} Slide show | Slide show Starts or stops the slide show - a mode when the program changes the pictures automatically after a certain interval, and in the specified direction (see below).
{[%hh_ii:SlideShowBackward]} Slide show | Forward Sets the slide show direction to Forward (from the first picture to the last one). In contrast with the setting value, this option affects only the current viewing session.
{[%hh_ii:SlideShowRandom]} Slide show | Random Sets the slide show direction to Random. In contrast with the setting value, this option affects only the current viewing session.
{[%hh_ii:SlideShowForward]} Slide show | Backward Sets the slide show direction to Backward (from the last picture to the first one). In contrast with the setting value, this option affects only the current viewing session.
{[%hh_ii:SlideShowCyclic]} Slide show | Cyclic slide show If this option is on, the slide show loops indefinitely. Otherwise it ends up once the last, or first picture is reached (it depends on the slide show direction). When the slide show direction is set to Random, the slide show never terminates automatically. In contrast with the setting value, this option affects only the current viewing session.
Slide show | Slide show interval Specifies the time, in seconds, during which to display a picture. In contrast with the setting value, this option affects only the current viewing session.
{[%hh_ii:Rotate0]} Transformations | No rotation Cancels the rotation for the current picture. Doesn't affect the next display of this picture. To apply the rotation permanently, use the Store transformations command.
{[%hh_ii:Rotate90]} Transformations | Rotate CW by 90° Rotates the current picture clockwise by 90°. Doesn't affect the next display of this picture. To apply the rotation permanently, use the Store transformations command.
{[%hh_ii:Rotate180]} Transformations | Rotate by 180° Rotates the current picture by 180°. Doesn't affect the next display of this picture. To apply the rotation permanently, use the Store transformations command.
{[%hh_ii:Rotate270]} Transformations | Rotate CCW by 90° Rotates the current picture counter-clockwise by 90° (or clockwise by 270°). Doesn't affect the next display of this picture. To apply the rotation permanently, use the Store transformations command.
{[%hh_ii:FlipHorz]} Transformations | Flip horizontally Applies a horizontal flip (along the vertical axis) to the current picture. Doesn't affect the next display of this picture. To apply the flip permanently, use the Store transformations command.
{[%hh_ii:FlipVert]} Transformations | Flip vertically Applies a vertical flip (along the horizontal axis) to the current picture. Doesn't affect the next display of this picture. To apply the flip permanently, use the Store transformations command.
{[%hh_ii:StoreTransform]} Transformations | Store transformations Stores all the transformations applied by the above items of this submenu, to the displayed pictures. The picture afterwards will be displayed having the same transformations applied.
{[%hh_ii:Props]} Program settings Opens the Program settings dialog, the View mode settings page.
