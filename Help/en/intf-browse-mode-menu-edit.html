{[%hh_hdr_en:Browse mode menu commands :: Edit menu]}

This menu contains the photo album editing commands, and also commands for switching to the View mode.
Menu Item Description
{[%hh_ii:Undo]} Undo operation Rolls back the changes to the photo album caused by indicated (the most recent) operation.
{[%hh_ii:UndoHistory]} Undo history Contains the list of changes made allowing you to roll back a number of sequential changes at once.
{[%hh_ii:Cut]} Cut Removes highlighted pictures and places them into the clipboard (you can add them later to another picture group).
{[%hh_ii:Copy]} Copy Copies highlighted pictures to the clipboard.
{[%hh_ii:Paste]} Paste Adds pictures contained in the clipboard (if any) to the current group. Obviously, you can only add new pictures to a group so any duplicate is skipped. If this is specified in the program settings, PhoA notifies you of number of the pictures actually pasted.
{[%hh_ii:NewGroup]} Add group Adds a new picture group. A group is created as a child to the group currently highlighted. Thus, when the photo album node is highlighted, a group is created in the photo album's root directory. The default group name is "New group", you change it by editing it and pressing Enter.
{[%hh_ii:NewPic]} Add pictures... Invokes the Add pictures wizard allowing you to add pictures (i. e. files) to the photo album.
{[%hh_ii:Delete]} Delete
  • When group tree is active: remove the current group.
  • When thumbnail window is active: remove selected thumbnails.
{[%hh_ii:Edit]} Edit...
  • When the photo album is selected in the group tree (the root node when picture groups are displayed), this items allows you to edit photo album's description or change thumbnail settings in the Photo album properties dialog.
  • When a photo album picture group is selected, this item allows you to rename it, or edit its description (you also can rename a group by clicking once on a selected group text, or by hitting the F2 key. To cancel edit, press Esc; to accept the new group name press Enter).
  • When a photo album view node is selected (the root node when a view is displayed), this item shows the View properties dialog.
  • When thumbnail window is active, this item displays the Picture properties dialog allowing you to view or modify selected picture properties.
{[%hh_ii:Sort]} Sort pictures... Arrange (sort) pictures in the current group, or in all photo album groups using the Sort pictures dialog.
{[%hh_ii:SelectAll]} Select all Selects all the pictures in the current group.
{[%hh_ii:SelectNone]} Select none Deselects all the selected pictures.
{[%hh_ii:ViewMode]} View Enters the program into the View mode. You can also enter the View mode by double-clicking a picture, or pressing Enter when thumbnail window is active.
{[%hh_ii:SlideShow]} View slide show Enters the View mode and turns the slide show on.
